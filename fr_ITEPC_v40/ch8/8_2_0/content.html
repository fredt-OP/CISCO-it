8.2 Description des types de réseau existants  

Les réseaux de données évoluent en permanence en termes de complexité, d’usage et de conception. En matière de terminologie, les différents types de réseau sont dotés de noms descriptifs. Un réseau informatique est ainsi identifié par les caractéristiques suivantes :

  • la zone qu’il dessert ;
  • la méthode de stockage des données ;
  • la manière dont les ressources sont gérées ;
  • l’organisation du réseau ;
  • le type de périphériques réseau utilisés ;
  • le type de support utilisé pour connecter les périphériques.

  • décrire un réseau local ;

  • expliquer le fonctionnement des réseaux d’égal à égal ;
  • expliquer le fonctionnement des réseaux client-serveur.
