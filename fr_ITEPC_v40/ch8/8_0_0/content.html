8.0 Introduction  

Ce chapitre contient une présentation générale des objectifs, principes et normes associés aux réseaux. Il abordera les types de réseaux suivants :

  • réseau local ;
  • réseau étendu ;
  • réseau local sans fil.

Ce chapitre vous permettra également de découvrir les différents types de topologie de réseau, de protocole et de modèle logique, ainsi que les équipements matériels requis pour la création d’un réseau. Il abordera également la configuration, le dépannage et la maintenance préventive. Vous y trouverez des informations sur les logiciels réseau, les méthodes de communication et les relations entre équipements matériels.

À la fin de ce chapitre, vous parviendrez à :

  • expliquer les principes de la mise en réseau ;
  • décrire les types de réseau existants ;
  • décrire les principaux concepts et technologies de mise en réseau ;
  • décrire les composants physiques d’un réseau ;
  • décrire les différentes topologies et architectures d’un réseau local ;
  • identifier les organismes de normalisation ;
  • identifier les normes Ethernet ;
  • expliquer les modèles de données OSI et TCP/IP ;
  • décrire comment configurer une carte réseau et un modem ;
  • identifier le nom, les fonctions et les caractéristiques des autres technologies de connectivité ;
  • identifier et appliquer des techniques courantes de maintenance préventive pour les réseaux ;
  • dépanner un réseau.
