8.4 Description des composants physiques d’un réseau  

De nombreux périphériques peuvent être utilisés pour assurer la connectivité d’un réseau. Le périphérique choisi dépend du nombre de systèmes à connecter, du type de connexion qu’ils utilisent et de leur vitesse de fonctionnement. Voici les périphériques réseau les plus courants :

  • ordinateurs ;
  • concentrateurs ;
  • commutateurs ;
  • routeurs ;
  • points d’accès sans fil.

Les composants physiques d’un réseau sont nécessaires pour transférer les données entre ces périphériques. Les caractéristiques du support déterminent où et comment ces composants sont utilisés. Voici les supports réseau les plus courants :

  • câble à paires torsadées ;
  • câble en fibre optique ;
  • ondes radioélectriques.

À la fin de cette section, vous parviendrez à :

  • identifier les noms, objectifs et caractéristiques des périphériques réseau ;
  • identifier les noms, objectifs et caractéristiques des câbles réseau courants.
