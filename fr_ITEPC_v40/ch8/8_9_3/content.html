8.9 Description de la configuration d’une carte réseau et d’un modem  
  8.9.3 Description de l’installation d’un modem  

Un modem est un périphérique électronique qui transfère des données entre deux ordinateurs, en utilisant des signaux analogiques transmis sur une ligne téléphonique. La Figure 1 présente quelques exemples de modem. Le modem convertit les données numériques en signaux analogiques, pour la transmission. Le modem récepteur reconvertit ces signaux analogiques en données numériques, interprétables par l’ordinateur. Le processus de conversion des signaux analogiques en numérique, et inversement, est appelé modulation/démodulation. Les transmissions par modem sont très précises, malgré le fait que les lignes téléphoniques puissent être bruyantes (déclics, électricité statique et autres).

Un modem interne se branche sur le logement d’extension de la carte mère. Pour configurer un modem, vous devez placer les cavaliers de manière à sélectionner les adresses IRQ et E/S. Aucune configuration n’est nécessaire pour les modems Plug and Play, qui peuvent être installés uniquement sur les cartes mères prenant en charge le matériel Plug and Play. Un modem branché sur un port série non encore utilisé doit faire l’objet d’une configuration. En outre, les pilotes logiciels fournis avec le modem doivent être installés pour que celui-ci fonctionne correctement. Les pilotes de modem s’installent de la même manière que les pilotes de carte réseau.

Les modems externes se connectent à l’ordinateur via les ports série et USB.

Lorsqu’un ordinateur utilise le réseau téléphonique public pour communiquer, on parle de réseau par ligne commutée (DUN, Dial-Up Networking ). Les modems communiquent les uns avec les autres par signaux de tonalité audio. Cela signifie que les modems sont capables de dupliquer les caractéristiques de numérotation d’un téléphone. La fonction DUN crée une connexion par protocole PPP (Point-to-Point) entre deux ordinateurs, via une ligne téléphonique.

Une fois la connexion de ligne établie, une séquence d’« échange bidirectionnel » est exécutée entre les deux modems et les ordinateurs. Cette séquence consiste en une série de brèves communications, établies entre les deux systèmes. Ces communications permettent de déterminer si les deux modems et l’ordinateur sont prêts pour l’échange de données. Les modems commutés envoient les données via la ligne téléphonique série, sous la forme d’un signal analogique. Les signaux analogiques changent progressivement et continuellement ; ils peuvent donc être représentés sous forme d’ondes. Dans ce système, les signaux numériques sont représentés par des 1 et des 0. Les signaux numériques doivent être convertis en ondes pour pouvoir circuler sur les lignes téléphoniques. Ils sont ensuite reconvertis au format numérique (1 et 0) par le modem récepteur, afin que l’ordinateur de destination puisse traiter les données.

Commandes AT
Tous les modems ont besoin d’un logiciel pour contrôler les sessions de communication. La plupart des logiciels de modem utilisent le jeu de commandes Hayes. Ce jeu de commandes est basé sur un ensemble d’instructions qui commencent toujours par les caractères d’attention (AT), suivis des caractères de commande. On parle alors de « commandes AT ». Le jeu de commandes AT est présenté dans la Figure 2.

Les commandes AT servent à contrôler le modem. Le jeu de commandes AT est utilisé pour émettre des instructions numéroter, raccrocher, réinitialiser, etc., pour le modem. La plupart des manuels d’utilisation fournis avec les modems contiennent une liste complète des commandes AT.

Le code Hayes standard à composer est ATDxxxxxxx. Les chaînes AT ne contiennent généralement aucun espace. Si un espace est inséré malgré tout, la plupart des modems l’ignorent. Le « x » correspond au numéro composé. Les numéros contiennent sept chiffres en appel local et 11 en appel longue distance. Le « W » indique que le modem attend une ligne extérieure, si nécessaire, pour obtenir la tonalité avant de continuer. Parfois, un « T » est ajouté, pour indiquer une numérotation au clavier, ou un « P » pour indiquer une numérotation au cadran.
