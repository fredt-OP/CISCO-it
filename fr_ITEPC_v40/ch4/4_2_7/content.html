4.2 Identification des étapes de la procédure de dépannage  
  4.2.7 Conclusion avec le client  

Une fois l’ordinateur réparé, terminez la procédure de dépannage en concluant avec le client. Communiquez au client le problème et la solution, verbalement et dans toute la documentation. La Figure 1 indique les étapes à suivre lorsque vous avez terminé une réparation et concluez avec le client.

Vérifiez la solution avec le client. Si le client est disponible, montrez-lui comment la solution a résolu le problème de l’ordinateur. Demandez au client de tester la solution et de tenter de reproduire le problème. Lorsque le client peut vérifier que le problème a été résolu, vous pouvez compléter la documentation de la réparation dans l’ordre de travail et dans votre journal. La documentation doit comprendre les informations suivantes :

  • description du problème ;
  • étapes suivies pour résoudre le problème ;
  • composants utilisés pour la réparation.
