13.3 Sélection des composants d’ordinateurs portables  
  13.3.3 Sélection de périphériques de stockage  

Les périphériques de stockage sont des CRU, sauf si une garantie nécessite une assistance technique. Il existe plusieurs options lors de l’ajout, du remplacement ou de la mise à niveau d’un périphérique de stockage pour un ordinateur portable :

  • Disque dur USB externe
  • Disque dur Firewire
  • Graveur de DVD/CD

Le format d’un périphérique de stockage interne de type disque dur est plus petit pour un ordinateur portable que pour un ordinateur de bureau. Cependant, un disque plus petit peut avoir une plus grande capacité de stockage.

Un disque dur USB externe se connecte à un ordinateur portable via le port USB. L’IEEE 1394 est un autre type de disque dur externe, qui se connecte sur le port Firewire. Un ordinateur portable détecte automatiquement le branchement d’un disque dur externe sur un port USB ou Firewire.

Un lecteur de DVD/CD RW est un lecteur optique capable de lire et d’écrire des données depuis et vers un CD, et de lire des données depuis un DVD. C’est un moyen pratique de créer des sauvegardes et d’archiver des données. Les deux types les plus courants de CD et DVD inscriptibles sont les CD et DVD inscriptibles (R) et réinscriptibles (RW).

Avant d’acheter un nouveau disque dur interne ou externe, consultez la documentation de l’ordinateur portable ou le site Web du fabricant concernant la compatibilité. La documentation contient souvent des Foires aux questions (FAQ) qui peuvent être utiles. Il est également important de consulter sur Internet les problèmes connus des composants de l’ordinateur portable.
