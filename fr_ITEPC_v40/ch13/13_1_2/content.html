13.1 Description des méthodes de communication sans fil pour les ordinateurs et périphériques portables  
  13.1.2 Description de la technologie infrarouge  

La technologie infrarouge (IR) sans fil est une technologie de faible puissance et à faible portée. L’IR transmet les données par le biais de diodes électroluminescentes (LED) et reçoit les données via des photodiodes.

Les réseaux IR sans fil ne sont généralement pas réglementés. Cependant, l’IrDA (Infrared Data Association) définit les spécifications de la communication IR sans fil. Reportez-vous à la Figure 1 pour connaître les caractéristiques IR courantes.

Il existe quatre types de réseaux IR :

  • visibilité directe : le signal est transmis uniquement si la visibilité entre les périphériques est claire et sans obstacle ;
  • diffusion : le signal rebondit sur les plafonds et les murs ;
  • réflectif : le signal est envoyé à un émetteur-récepteur optique, puis il est redirigé vers le périphérique de réception ;
  • télépoint optique à large bande : la transmission est capable de gérer le multimédia haute qualité.

Les réseaux infrarouges sont parfaits pour connecter des ordinateurs portables aux types de périphériques suivants, lorsqu’ils sont proches :

  • projecteur multimédia ;
  • assistant numérique personnel ;
  • imprimante ;
  • télécommande ;
  • souris sans fil ;
  • clavier sans fil.

La configuration de périphériques IR est assez simple. De nombreux périphériques IR se connectent sur le port USB d’un ordinateur portable ou de bureau. Lorsque l’ordinateur détecte le nouveau périphérique, Windows XP installe les pilotes adéquats, tel qu’illustré à la Figure 2. L’installation est similaire à la configuration d’une connexion de réseau local.

L’IR est une solution pratique de connexion à faible portée, mais elle a certaines limites :

  • la lumière infrarouge ne peut pas traverser les plafonds ou les murs ;
  • les signaux IR sont sensibles aux interférences et à la dilution provenant de sources lumineuses puissantes, comme la lumière fluorescente ;
  • les périphériques IR de diffusion peuvent se connecter sans visibilité directe, mais le débit du transfert des données est plus faible et les distances de fonctionnement plus courtes ;
  • les distances IR doivent être inférieures ou égales à 1 mètre en cas de communication entre ordinateurs.
