13.4 Description des procédures de maintenance préventive pour les ordinateurs portables  
  13.4.2 Explication de la gestion du contrôle de la version des données entre les ordinateurs de bureau et portables  

Il est important de gérer correctement vos fichiers et dossiers de données. Les procédures de restauration et de récupération, ainsi que les sauvegardes, sont plus efficaces si les données sont organisées.

Windows XP propose un emplacement par défaut, parfois disponible sous la forme d’une icône sur le bureau, appelé Mes documents. Vous pouvez utiliser Mes documents pour créer une structure de dossiers et stocker des fichiers.

Lorsque vous déplacez des fichiers d’un ordinateur portable vers un ordinateur de bureau, créez d’abord une structure de dossiers similaire aux deux emplacements. Les fichiers peuvent être transférés par le biais d’un réseau, d’un disque optique ou d’un lecteur portable.

Vous devez veiller à ce que les données copiées depuis un ordinateur n’écrasent pas accidentellement les données sur l’autre ordinateur. Lorsque vous copiez un fichier vers un dossier de destination, vous pouvez rencontrer le message « Confirmer le remplacement du fichier », tel qu’illustré à la Figure 1. Ce message indique que Windows XP interrompt le processus de copie jusqu’à ce que vous choisissiez de remplacer ou non le fichier original par le fichier transféré. En cas de doute, sélectionnez « Non ». Pour savoir quel fichier garder, comparez les dates et la taille du fichier. Vous pouvez également ouvrir les fichiers pour visualiser leur contenu.

REMARQUE : aucun système d’exploitation n’autorise la coexistence de fichiers portant le même nom dans un même dossier.

ATTENTION : veillez à ne pas « couper » par inadvertance un fichier de son emplacement d’origine si vous souhaitez seulement le « copier ».
