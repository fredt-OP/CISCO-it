6.4 Configuration des ordinateurs portables  
  6.4.1 Description de la configuration des paramètres d’alimentation  

L’une des caractéristiques les plus appréciées des ordinateurs portables est leur fonctionnement sur batterie. Cela permet de les utiliser dans des situations où une alimentation CA n’est pas disponible ou peu pratique. Les progrès en matière de gestion de l’alimentation et de technologie des batteries augmentent la durée pendant laquelle les utilisateurs d’ordinateurs portables peuvent rester déconnectés de l’alimentation secteur. Les batteries actuelles permettent une autonomie de 2 à 10 heures avant de devoir être rechargées. La gestion de l’alimentation d’un ordinateur portable, au moyen d’une configuration correcte des paramètres appropriés, est particulièrement importante pour assurer une utilisation efficace de la batterie.  

Les normes ACPI (Advanced Configuration and Power Interface) établissent une interface entre le matériel et le système d’exploitation, qui permet aux techniciens de créer des modèles de gestion de l’alimentation afin d’obtenir les meilleures performances de l’ordinateur. Les normes ACPI peuvent être appliquées à la plupart des ordinateurs, mais elles sont particulièrement importantes pour la gestion de l’alimentation des ordinateurs portables. Cliquez sur les états de gestion de l’alimentation dans la Figure 1 pour obtenir plus d’informations sur chaque état.

Les techniciens doivent fréquemment configurer les paramètres de l’alimentation en modifiant les paramètres du BIOS. Cela influence les conditions suivantes :

  • états système ;
  • modes batterie et CA ;
  • gestion thermique ;
  • gestion de l’alimentation du bus UC PCI (interconnexion de composants périphériques).

La Figure 2 présente un exemple de paramètres d’alimentation du BIOS. 

REMARQUE : lors d’un fonctionnement sous Windows XP, le mode de gestion de l’alimentation ACPI doit être activé dans le BIOS pour permettre au SE de configurer tous les états de gestion de l’alimentation. 

REMARQUE : il n’existe pas de nom normalisé pour chaque état de gestion de l’alimentation. Les fabricants peuvent utiliser différents noms pour le même état. 

Voici les étapes à suivre pour vérifier les paramètres ACPI dans le BIOS :

  1. Entrez dans la configuration du BIOS en appuyant sur la touche ou sur la combinaison de touches appropriée lorsque l’ordinateur s’amorce. En général, il s’agit de la touche Suppr ou F2 , mais plusieurs autres options sont possibles.
  2. Recherchez l’élément de menu permettant d’accéder aux paramètres de gestion de l’alimentation.
  3. Utilisez les touches appropriées pour activer le mode ACPI.
  4. Enregistrez et quittez la configuration du BIOS.

REMARQUE : ces étapes sont communes à la plupart des ordinateurs portables et doivent être utilisées comme une indication uniquement. Vous devez vérifier le manuel de votre ordinateur portable pour obtenir les paramètres de configuration qui lui sont propres. 

La fonction Options d’alimentation sous Windows XP vous permet de réduire la consommation d’énergie de nombreux périphériques ou de l’intégralité du système. Utilisez la fonction Options d’alimentation pour gérer l’alimentation des éléments suivants :

  • disque dur ;
  • écran ;
  • modes arrêt, veille prolongée et veille ;
  • avertissements de batterie faible.

Configuration de la gestion de l’alimentation sous Windows XP
Vous pouvez régler la gestion de l’alimentation au moyen de la fonction Options d’alimentation du Panneau de configuration. La fonction Options d’alimentation affiche uniquement les options qui peuvent être contrôlées.

Remarque : la fonction Options d’alimentation détecte automatiquement les périphériques propres à votre ordinateur. Le contenu de la fenêtre Options d’alimentation peut donc varier selon le matériel détecté. 

Pour configurer vos paramètres d’alimentation, cliquez sur :
Démarrer > Panneau de configuration > Options d’alimentation

Modèles d’alimentation
Les modèles d’alimentation constituent un ensemble de paramètres qui gèrent l’utilisation de l’alimentation par l’ordinateur. Le disque dur et l’écran consomment beaucoup d’énergie. Ils peuvent être configurés sous l’onglet Modèles d’alimentation. 

Lorsque vous ouvrez la fenêtre Options d’alimentation, vous remarquerez que Windows XP propose des modèles d’alimentation prédéfinis. Il s’agit de valeurs par défauts, établies lors de l’installation de Windows XP. Vous pouvez utiliser les modèles d’alimentation par défaut ou créer des modèles personnalisés, basés sur vos exigences professionnelles spécifiques. Personnaliser l’alimentation de l’écran et du disque dur
Dans ce scénario, l’étudiant a besoin d’accéder à l’écran, mais il accède rarement au disque dur. Par exemple, il se sert d’un ordinateur portable pour des recherches sur Internet, mais il ne crée et n’enregistre pas fréquemment de fichiers.

Gestion de l’alimentation du disque dur
Le disque dur est l’un des périphériques qui consomment le plus d’énergie sur un ordinateur portable. Dans notre exemple, l’utilisateur n’accède pas souvent au disque dur. La durée « Arrêt des disques durs » est définie sur 1 heure lorsque l’ordinateur fonctionne sur secteur et sur 3 minutes en cas de « Fonctionnement sur batteries ».

Vous décidez que les valeurs par défaut pour les modes de veille et de veille prolongée sont acceptables et qu’aucune modification est nécessaire. Les modèles d’alimentation peuvent être enregistrés sous un nom personnalisé. Cela permet à l’utilisateur de revenir facilement aux paramètres par défaut. Dans cet exemple, enregistrez les paramètres du modèle d’alimentation sous « Rechercher », comme illustré à la Figure 4.

Mise en état de veille ou de veille prolongée
Si vous ne souhaitez pas éteindre complètement un ordinateur portable, vous disposez de deux options : veille et veille prolongée.

  • Veille : les documents et les applications sont enregistrés dans la mémoire vive, ce qui permet à l’ordinateur de revenir rapidement en fonction.
  • Veille prolongée : les documents et les applications sont enregistrés dans un fichier temporaire sur le disque dur, si bien que l’ordinateur revient en fonction plus lentement que depuis l’état de veille.

La Figure 5 illustre la fonction Veille prolongée activée dans les propriétés Options d’alimentation. 

Réglage des avertissements de batterie faible
Sous Windows XP, vous pouvez définir des avertissements de batterie faible. Deux niveaux sont possibles : alerte de batterie faible et alerte de batterie critique. L’alerte de batterie faible vous avertit que le niveau de la batterie est faible. L’alerte de batterie critique force une veille, une veille prolongée ou un arrêt, comme illustré à la Figure 6.
