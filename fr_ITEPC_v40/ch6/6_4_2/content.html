6.4 Configuration des ordinateurs portables  
  6.4.2 Description de l’installation et le retrait sécurisés de composants d’ordinateurs portables  

Il peut être nécessaire de remplacer de nombreux composants sur un ordinateur portable. Vérifiez toujours que vous disposez du composant de rechange correct et des outils appropriés, conformément aux recommandations du fabricant. Certains composants sont remplaçables à chaud, c’est-à-dire qu’ils peuvent être retirés et remplacés alors que l’ordinateur est sous tension. Voici quelques composants qui peuvent avoir besoin d’être remplacés :

  • batterie ;
  • lecteur optique ;
  • disque dur ;
  • mémoire :
  • PC Cards.

La Figure 1 présente un exemple d’ordinateur portable. 

REMARQUE : chaque fabricant d’ordinateur portable utilise ses propres procédures de pose et de dépose. Consultez le manuel de l’ordinateur portable pour obtenir des informations d’installation spécifiques, respectez les consignes de sécurité en matière d’installation et les précautions à prendre contre les décharges électrostatiques. 

ATTENTION : vous devez toujours débrancher l’ordinateur portable et retirer la batterie avant d’installer ou de retirer des composants qui ne sont pas remplaçables à chaud. 

Étapes de remplacement de la batterie [Figure 2]
Retirez la batterie de la baie de batterie :

  1. placez le verrou de la batterie en position déverrouillée ;
  2. maintenez le levier de dégagement en position déverrouillée et retirez la batterie.

Installez la batterie dans la baie de batterie :

  1. insérez la batterie ;
  2. vérifiez que les deux verrous de batterie sont verrouillés.

Étapes de remplacement du lecteur optique [Figure 3]
Retirez le lecteur de DVD/CD-RW :

  1. appuyez sur le bouton pour ouvrir le tiroir et retirez tout support éventuellement présent ; fermez le tiroir ;
  2. faites glisser le loquet pour dégager le levier qui maintient le lecteur ;
  3. tirez sur le levier pour exposer le lecteur. Retirez le lecteur.

Installez le lecteur de DVD/CD-RW :

  1. insérez le lecteur à fond ;
  2. poussez le levier vers l’intérieur.

Étapes de remplacement du disque dur [Figure 4]
Retirez le disque dur :

  1. retirez la vis, située sous l’ordinateur portable, qui maintient le disque dur en place ;
  2. faites glisser l’ensemble vers l’extérieur ; retirez l’ensemble du disque dur.
  3. retirez la plaque du disque dur.

Installez le disque dur :

  1. fixez la plaque sur le disque dur ;
  2. faites glisser le disque dur dans son compartiment ;
  3. vissez la vis, située sous l’ordinateur portable, qui maintient le disque dur en place.

Étapes de remplacement de la mémoire d’extension [Figure 5]
La mémoire d’extension d’un ordinateur portable est également appelée SODIMM (small outline dual in-line memory module, module mémoire de petit format à double rangée). Retirez le SODIMM existant s’il n’y a pas de logements disponibles pour le nouveau SODIMM :

  1. retirez la vis pour exposer le SODIMM ;
  2. appuyez vers l’extérieur sur les pinces qui maintiennent les côtés du SODIMM ;
  3. soulevez le SODIMM pour le dégager du logement et retirez-le.

Installez le SODIMM :

  1. alignez l’encoche à un angle de 45° ;
  2. appuyez doucement jusqu’à ce que les pinces se verrouillent :
  3. replacez le couvercle et vissez la vis.

Étapes de remplacement des cartes d’extension PC [Figure 6]
Retirez la carte d’extension PC :

  1. appuyez sur le bouton d’éjection supérieur pour dégager la carte d’extension PC.

REMARQUE : deux boutons sont présents. Le bouton bleu inférieur éjecte la PC Card de type II.

Installez la carte d’extension PC :

  1. appuyez sur le bouton bleu, vers l’intérieur ;
  2. insérez la carte d’extension PC dans le logement Express.

ATTENTION : sur certains ordinateurs portables, les PC Card, lecteur optique et périphériques USB sont remplaçables à chaud. Toutefois, le disque dur interne, la mémoire vive et la batterie ne sont PAS remplaçables à chaud. 

Étapes de retrait des périphériques remplaçables à chaud

  1. Dans la zone de notification Windows, cliquez avec le bouton gauche sur l’icône Retirer le périphérique en toute sécurité, pour être sûr que le périphérique n’est pas utilisé.
  2. Cliquez avec le bouton gauche sur le périphérique que vous souhaitez retirer. Un message s’affiche pour vous informer que vous pouvez retirer le périphérique en toute sécurité.
  3. Retirez de l’ordinateur le périphérique remplaçable à chaud.

  Exercice facultatif : Composants et périphériques de l’ordinateur portable virtuel

La configuration système requise pour l’ordinateur portable virtuel inclut une mémoire vive de 512 Mo minimum et le système d’exploitation Windows 2000 ou XP.
