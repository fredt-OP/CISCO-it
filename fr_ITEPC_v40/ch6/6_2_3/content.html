6.2 Identification et description des composants d’un ordinateur portable  
  6.2.3 Description des composants se trouvant sur une station d’accueil d’ordinateur portable  

Une station de base est un équipement alimenté par secteur et relié à des périphériques de bureau. Lorsque vous branchez l’ordinateur portable sur la station de base, vous disposez de l’alimentation secteur et d’un accès pratique aux périphériques connectés.

Il existe deux types de stations de base : les stations d’accueil et les réplicateurs de ports. Elles sont utilisées dans le même but. Toutefois, les réplicateurs de ports sont généralement plus petits que les stations d’accueil et ils ne disposent pas de haut-parleurs ni de logements pour l’interconnexion de composants périphériques (PCI). Les Figures 1 à 3 présentent une station d’accueil.

Cliquez sur les trois zones mises en évidence dans la Figure 1 pour obtenir plus d’informations sur les composants situés sur le dessus de la station d’accueil :

  1. bouton d’alimentation ;
  2. bouton d’éjection ;
  3. connecteur d’ordinateur portable.

Certaines stations d’accueil comportent les compartiments de lecteur et les ports suivants pour fournir des fonctions supplémentaires :

  • parallèle ;
  • USB ;
  • Ethernet ;
  • vidéo ;
  • son.

L’arrière de la station d’accueil contient les ports et les connecteurs utilisés pour relier celle-ci à des périphériques de bureau comme une souris, un moniteur ou une imprimante. Une aération est également nécessaire pour expulser l’air chaud de la station d’accueil. Cliquez sur les quinze zones mises en évidence dans la Figure 2 pour obtenir plus d’informations sur les composants situés à l’arrière de la station d’accueil :

  1. grille de ventilation ;
  2. connecteur d’alimentation secteur (CA) ;
  3. logement PC Card/ExpressCard ;
  4. port VGA ;
  5. port DVI ;
  6. connecteur Entrée ligne ;
  7. connecteur de casque ;
  8. port USB ;
  9. port de souris ;
  10. port de clavier ;
  11. connecteur de lecteur de disquette externe ;
  12. port parallèle ;
  13. port série ;
  14. port modem ;
  15. port Ethernet.

Utilisez la serrure pour sécuriser l’ordinateur portable dans la station d’accueil. Cliquez sur les zones mises en évidence dans la Figure 3 pour obtenir plus d’informations sur la serrure située à droite de la station d’accueil.
