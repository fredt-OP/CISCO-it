14.3 Description du partage d’une imprimante et d’un scanner sur un réseau  
  14.3.2 Description de l’installation du logiciel et des pilotes de l’imprimante réseau sur un ordinateur  

Windows XP permet aux utilisateurs d’ordinateurs de partager leurs imprimantes avec d’autres utilisateurs sur le réseau. On distingue deux étapes :

  1. Configurez l’ordinateur relié à l’imprimante pour partager celle-ci avec d’autres utilisateurs du réseau.
  2. Configurez l’ordinateur des utilisateurs pour reconnaître l’imprimante partagée et pouvoir imprimer.

Pour configurer l’ordinateur relié à l’imprimante, afin qu’il accepte des travaux d’impression d’autres utilisateurs du réseau, procédez comme suit :

  1. Choisissez Démarrer > Panneau de configuration > Imprimantes et autres périphériques > Imprimantes et télécopieurs.
  2. Sélectionnez l’imprimante à partager.
  3. La case Tâches de l’imprimante s’affiche sur la gauche. Sélectionnez Partager cette imprimante.
  4. La boîte de dialogue Propriétés de l’imprimante s’affiche pour cette imprimante. Sélectionnez l’onglet Partage. Sélectionnez Partager cette imprimante et entrez le nom de partage choisi. C’est le nom que portera l’imprimante pour les autres utilisateurs.
  5. Vérifiez que le partage a réussi. Revenez dans le dossier Imprimantes et télécopieurs : vous remarquez que l’icône de l’imprimante repose désormais sur une main, tel qu’illustré à la Figure 1. Cela signifie que l’imprimante est maintenant une ressource partagée.

D’autres utilisateurs, qui peuvent désormais se connecter à l’imprimante partagée, risquent de ne pas disposer des pilotes nécessaires. Ces autres utilisateurs peuvent également utiliser des systèmes d’exploitation différents de celui de l’ordinateur qui héberge l’imprimante partagée. Windows XP peut télécharger automatiquement les bons pilotes pour ces autres utilisateurs. Cliquez sur le bouton Pilotes supplémentaires pour sélectionner les systèmes d’exploitation pouvant être utilisés par les autres utilisateurs. Lorsque vous fermez cette boîte de dialogue en cliquant sur OK, Windows XP demande à obtenir ces pilotes supplémentaires. Si tous les autres utilisateurs utilisent également Windows XP, vous n’avez pas besoin de cliquer sur le bouton Pilotes supplémentaires.

Connexion d’autres utilisateurs
D’autres utilisateurs du réseau peuvent désormais se connecter à cette imprimante en procédant comme suit :

  1. Choisissez Démarrer > Panneau de configuration > Imprimantes et autres périphériques > Ajout d’imprimante.
  2. L’assistant Ajout d’imprimante apparaît. Cliquez sur Suivant.
  3. Sélectionnez Une imprimante réseau ou une imprimante connectée à un autre ordinateur, tel qu’illustré à la Figure 2. Cliquez sur Suivant.
  4. Saisissez le nom de l’imprimante ou recherchez-la sur le réseau grâce au bouton Suivant. Une liste d’imprimantes partagées s’affiche.
  5. Après avoir sélectionné l’imprimante, un port virtuel d’imprimante est créé et affiché dans la fenêtre Ajout d’imprimante. Les pilotes d’impression nécessaires sont téléchargés depuis le serveur d’impression et installés sur l’ordinateur. L’Assistant termine alors l’installation.
