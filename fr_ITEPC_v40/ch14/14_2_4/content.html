14.2 Installation et configuration en local d’une imprimante et d’un scanner  
  14.2.4 Vérification des fonctionnalités  

L’installation d’un périphérique n’est pas terminée tant que vous n’avez pas testé toutes les fonctions du périphérique. Ce test inclut des tâches spéciales, comme les tâches suivantes :

  • impression de documents recto verso pour économiser du papier ;
  • utilisation de différents types de bacs d’alimentation pour les formats spéciaux de papier ;
  • modification des paramètres d’une imprimante couleur pour qu’elle imprime des brouillons de documents en noir et blanc ou en échelle de gris ;
  • impression en mode brouillon pour économiser de l’encre ;
  • modification de la résolution de numérisation d’un scanner pour faciliter l’affichage d’une image ;
  • modification d’images numérisées de documents enregistrés ;
  • utilisation d’une application de reconnaissance optique de caractères (OCR).

REMARQUE : les manuels électroniques et les sites Web d’assistance expliquent comment remédier aux bourrages papier, installer des cartouches d’encre et charger tout type de bac d’alimentation.

Test de l’imprimante
Il existe plusieurs méthodes pour imprimer une page de test :

  • utilisation de l’option Imprimer une page de test de l’imprimante ;
  • utilisation de l’option Imprimer une page de test sous Windows ;
  • utilisation de la fonction d’impression d’une application ;
  • envoi d’un fichier directement à une imprimante branchée en parallèle via la ligne de commande.

Pour tester une imprimante, imprimez d’abord une page de test à partir de l’imprimante, puis à partir de la fonction Propriétés de l’ordinateur ou d’une application. Vous vous assurez ainsi du bon fonctionnement de l’imprimante, de l’installation et du fonctionnement adéquats du logiciel pilote, ainsi que de la bonne communication entre l’imprimante et l’ordinateur.

Test du scanner
Testez le scanner en numérisant un document. Utilisez les boutons sur le périphérique pour la numérisation automatique. Ensuite, lancez des numérisations à partir du logiciel du scanner et assurez-vous que le logiciel ouvre la numérisation automatique. Si les images numérisées semblent être identiques à celle affichée à l’écran, l’installation est réussie.

Pour un périphérique tout-en-un, vous devez tester toutes les fonctions suivantes :

  • Fax : faxez vers un autre fax qui fonctionne.
  • Copie : créez une copie d’un document.
  • Numérisation : numérisez un document.
  • Impression : imprimez un document.
