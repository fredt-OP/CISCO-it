14.3 Description du partage d’une imprimante et d’un scanner sur un réseau  
  14.3.1 Description des types de serveurs d’impression  

Les serveurs d’impression permettent à plusieurs utilisateurs d’ordinateurs d’accéder à une même imprimante. Un serveur d’impression a trois fonctions. La première fonction vise à offrir un accès client aux ressources d’impression. La deuxième permet d’administrer les travaux d’impression, en les stockant dans une file d’attente jusqu’à ce que le périphérique d’impression soit prêt, puis en fournissant ou en gérant les informations d’impression à l’imprimante. La troisième permet d’offrir un retour aux utilisateurs, notamment l’envoi d’une notification lorsqu’un travail d’impression est terminé ou l’envoi de messages d’erreur lorsque quelque chose ne va pas.

En tant que technicien, vous devez choisir le type de serveur d’impression correspondant le mieux aux besoins du client. Il en existe trois types :

  • le serveur d’impression réseau ;
  • le serveur d’impression sur PC dédié ;
  • l’imprimante partagée sur un ordinateur.

Serveur d’impression réseau
Les serveurs d’impression réseau permettent à plusieurs utilisateurs sur un réseau d’accéder à une même imprimante. Un serveur d’impression réseau peut gérer l’impression réseau par le biais de connexions câblées ou sans fil. La Figure 1 illustre un serveur d’impression câblé. Vous devez tenir compte des avantages et inconvénients d’un serveur d’impression sur PC dédié avant toute installation :

  • L’utilisation d’un serveur d’impression réseau présente l’avantage suivant : le serveur accepte les travaux d’impression entrants de la part d’ordinateurs, puis libère les ordinateurs pour d’autres tâches. Le serveur d’impression est toujours disponible pour les utilisateurs, à la différence d’une imprimante partagée par l’ordinateur d’un utilisateur.
  • Le serveur d’impression réseau présente l’inconvénient de ne pas être toujours en mesure d’utiliser toutes les fonctions d’un périphérique tout-en-un.

Serveur d’impression sur PC dédié
Un serveur d’impression sur PC dédié est un ordinateur dédié à la gestion la plus efficace des travaux d’impression clients. Comme il gère les requêtes de plusieurs clients, le serveur d’impression est généralement l’un des ordinateurs les plus puissants du réseau. Les serveurs d’impression sur PC dédié peuvent gérer plusieurs imprimantes à la fois. Un serveur d’impression doit disposer des ressources nécessaires pour répondre aux requêtes des clients d’impression :

  • Processeur puissant : le serveur d’impression sur PC utilisant son processeur pour gérer et transférer les informations d’impression, il doit être suffisamment rapide pour gérer toutes les requêtes entrantes.
  • Espace disque suffisant : un serveur d’impression sur PC capture les travaux d’impression émis par les clients, les place dans une file d’attente d’impression et les envoie à l’imprimante au moment opportun. Pour cela, l’ordinateur doit disposer d’un espace de stockage suffisant pour supporter ces travaux jusqu’à leur achèvement.
  • Mémoire adaptée : le processeur et la mémoire vive du serveur gèrent l’envoi de travaux d’impression à une imprimante. Si la mémoire du serveur n’est pas suffisante pour gérer l’intégralité d’un travail d’impression, le disque dur doit envoyer le travail, ce qui est beaucoup plus lent.

Imprimante partagée sur un ordinateur
L’ordinateur d’un utilisateur disposant d’une imprimante connectée peut la partager avec d’autres utilisateurs sur le réseau. Windows XP rend ce processus rapide et facile. Dans un réseau domestique, cela signifie qu’un utilisateur peut imprimer des documents où qu’il se trouve dans la maison, en utilisant un ordinateur portable sans fil. Dans un réseau de petits bureaux, le partage d’une imprimante signifie qu’un même périphérique peut servir à plusieurs utilisateurs.

Le partage d’une imprimante à partir d’un ordinateur a également des inconvénients. L’ordinateur partageant l’imprimante utilise ses propres ressources pour gérer les travaux d’impression entrants pour l’imprimante. Si un utilisateur travaille sur l’ordinateur en même temps qu’un utilisateur imprime sur le réseau, l’utilisateur sur l’ordinateur peut constater une baisse des performances. De plus, l’imprimante n’est pas disponible si l’utilisateur redémarre ou éteint l’ordinateur connecté à l’imprimante partagée.
