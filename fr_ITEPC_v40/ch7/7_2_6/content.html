7.2 Description de la procédure d’installation et de configuration des imprimantes  
  7.2.6 Impression d’une page de test  

Après avoir installé une imprimante, vous devez imprimer une page de test pour vérifier que l’imprimante fonctionne correctement. La page de test confirme que le logiciel du pilote est installé, qu’il fonctionne correctement et que l’imprimante et l’ordinateur peuvent communiquer.

Impression d’une page de test
Pour imprimer manuellement une page de test, suivez le chemin :
Démarrer > Imprimantes et télécopieurs pour afficher le menu Imprimantes et télécopieurs.

Cliquez avec le bouton droit de la souris sur l’imprimante souhaitée, puis suivez le chemin :
Propriétés > onglet Général > Imprimer une page de test

Une boîte de dialogue s’ouvre et demande si la page a été imprimée correctement. Si la page n’a pas été imprimée, des fichiers d’aide intégrés vous aideront à résoudre le problème.

Impression depuis une application
Vous pouvez également tester une imprimante en imprimant une page de test depuis une application comme Bloc-notes ou WordPad. Pour accéder au Bloc-notes, suivez le chemin :
Démarrer > Programmes > Accessoires > Bloc-notes

Un document vide s’ouvre. Entrez du texte dans le document. Imprimez-le en suivant le chemin :
Fichier > Imprimer

Test d’une imprimante
Vous pouvez également tester une imprimante à partir de la ligne de commande. Seuls les fichiers ASCII, tels que les fichiers .txt ou .bat, peuvent être imprimés depuis la ligne de commande. Pour envoyer un fichier à l’imprimante depuis la ligne de commande, suivez le chemin :
Démarrer > Exécuter

La boîte de dialogue Exécuter s’affiche. Entrez cmd dans la boîte de dialogue Exécuter, puis cliquez sur OK.

Lorsque l’invite de la ligne de commande apparaît, entrez la commande suivante :
Print lefichier.txt

Test d’une imprimante depuis son panneau avant
La plupart des imprimantes comportent un panneau avant muni de touches de commandes qui vous permettent de produire des pages de test. Cette méthode d’impression vous permet de vérifier le fonctionnement de l’imprimante séparément de celui du réseau ou de l’ordinateur. Consultez le site Web ou la documentation du fabricant de l’imprimante pour savoir comment imprimer une page de test depuis le panneau avant.
