7.3 Description des types de scanners actuellement disponibles  
  7.3.1 Description des types de scanners, de leur résolution et de leurs interfaces  

Les scanners sont utilisés pour convertir des données ou des images imprimées à un format de données électroniques qu’un ordinateur peut ensuite conserver ou traiter comme requis. Une fois numérisée, l’image peut être enregistrée, modifiée et même envoyée par courriel, comme n’importe quel autre fichier. Bien que la plupart des scanners effectuent la même opération, plusieurs types de scanners différents sont disponibles, comme illustré à la Figure 1. Cliquez sur chaque type de scanner pour obtenir plus d’informations.

Comme pour les imprimantes, les fonctions, la qualité et la vitesse des scanners varient selon le type d’appareil. En général, les scanners créent une image RVB qui peut être convertie aux formats d’image courants tels que JPEG, TIFF, Bitmap et PNG. Une image RVB comporte trois canaux : rouge, vert et bleu. Les canaux RVB correspondent en général aux récepteurs de couleurs de l’œil humain et sont utilisés pour les affichages par ordinateur et pour les scanners d’images.

Certains scanners peuvent créer des documents texte en utilisant un logiciel de reconnaissance optique de caractères (OCR, Optical Character Recognition). Les logiciels de reconnaissance optique de caractères sont utilisés pour convertir une page imprimée numérisée en un texte, qui peut ensuite être modifié au moyen d’un logiciel de traitement de texte. La résolution d’un scanner est mesurée en points par pouce (ppp). Comme pour les imprimantes, plus le nombre de points par pouce est élevé, plus la qualité de l’image est bonne.

Le scanner et l’ordinateur doivent disposer d’interfaces compatibles pour permettre la communication de données entre eux. Les interfaces et les câbles utilisés pour les scanners sont généralement les mêmes que ceux utilisés pour les imprimantes, comme illustré à la Figure 2.
