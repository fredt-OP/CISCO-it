7.1 Description des types d’imprimantes actuellement disponibles  

En tant que technicien informatique, vous pouvez être amené à acheter, réparer ou entretenir une imprimante. Le client peut vous demander d’effectuer les tâches suivantes :

  • sélectionner une imprimante ;
  • installer et configurer une imprimante ;
  • dépanner une imprimante.

À la fin de cette section, vous parviendrez à :

  • décrire les caractéristiques et les possibilités des imprimantes ;
  • décrire les interfaces entre une imprimante et un ordinateur ;
  • décrire les imprimantes laser ;
  • décrire les imprimantes à impact ;
  • décrire les imprimantes à jet d’encre ;
  • décrire les imprimantes à encre solide ;
  • décrire d’autres types d’imprimantes.
