7.3 Description des types de scanners actuellement disponibles  
  7.3.4 Description des scanners portables  

Un scanner portable est un petit scanner portatif. Il est difficile de numériser une image correctement en utilisant un scanner portable. Vous devez passer soigneusement la tête du scanner sur le document à numériser. Comme pour un scanner à plat, les images numériques sont constituées des images recueillies par le scanner portable.

Si vous souhaitez numériser un document plus grand que la tête du scanner portable, vous devez effectuer plusieurs passages pour capturer l’image complète. Il peut être difficile de recréer numériquement l’image d’origine lorsqu’elle a été numérisée par plusieurs passages. Les images doivent être réunies pour former l’image unique du document numérisé.
