7.4 Description de la procédure d’installation et de configuration des scanners  

Lorsque vous achetez un scanner, les informations d’installation et de configuration sont généralement fournies par le fabricant. Un CD d’installation, qui comporte les pilotes, les manuels et le logiciel de diagnostic, est inclus avec le scanner. Le cas échéant, ces outils peuvent également être téléchargés depuis le site Web du fabricant.

À la fin de cette section, vous parviendrez à :

  • expliquer comment alimenter électriquement un scanner et le connecter ;
  • décrire comment installer et mettre à jour le pilote du périphérique ;
  • identifier les options de configuration et les paramètres par défaut.
