7.1 Description des types d’imprimantes actuellement disponibles  
  7.1.2 Description des interfaces entre une imprimante et un ordinateur  

Pour imprimer des documents, un ordinateur doit disposer d’une interface compatible avec l’imprimante. En général, les imprimantes se connectent aux ordinateurs particuliers au moyen d’une interface parallèle, USB ou sans fil. Les imprimantes d’entreprise peuvent se connecter à un réseau au moyen d’un câble réseau.

Série
Le transfert série de données est le déplacement de bits uniques d’information dans un seul cycle. Une connexion série peut être utilisée pour les imprimantes matricielles car elles ne requièrent pas un haut débit de transfert de données.

Parallèle
Le transfert parallèle de données est plus rapide que le transfert série. Il s’agit d’un déplacement de bits multiples d’information dans un seul cycle. Le chemin est alors plus large pour déplacer les informations vers l’imprimante ou à partir de celle-ci.

IEEE 1284 est la norme actuelle pour les ports parallèles d’imprimante. Les modes EPP (Enhanced Parallel Port) et ECP (Enhanced Capabilities Port) sont deux modes de fonctionnement de la norme IEEE 1284 permettant une communication bidirectionnelle.

SCSI
L’interface SCSI (Small Computer System Interface) utilise une technologie de communication parallèle pour atteindre des débits élevés de transfert de données.

USB
USB est une interface très répandue pour les imprimantes et d’autres périphériques. Sa configuration rapide et simple la rend très pratique. Les systèmes d’exploitation plus récents proposent une prise en charge USB Plug and Play. Lorsqu’un périphérique USB est ajouté à un système informatique prenant en charge le Plug and Play, le périphérique est automatiquement détecté et le processus d’installation du pilote est lancé.

FireWire
FireWire, également connu sous le nom de i.LINK ou IEEE 1394, est un bus de communication à haut débit, indépendant des plateformes. Ce bus connecte des périphériques numériques tels que les imprimantes numériques, les scanners, les appareils photo numériques et les disques durs.

FireWire permet à un périphérique, comme une imprimante, de se connecter à un ordinateur de façon transparente. Il permet également au périphérique d’être remplaçable à chaud. FireWire fournit une prise complète mâle et femelle unique, à laquelle peuvent se brancher jusqu’à 63 périphériques. Le débit de transfert de données de FireWire peut atteindre 400 Mbits/s.

Ethernet
Les imprimantes peuvent être partagées sur un réseau. La connexion d’une imprimante au réseau requiert un câblage compatible au réseau existant et au port de réseau installé dans l’imprimante. La plupart des imprimantes de réseau utilisent une interface RJ-45 pour se connecter au réseau.

Sans fil
L’impression sans fil est possible au moyen des technologies infrarouge, Bluetooth et Wi-Fi (Wireless Fidelity).

Pour une communication infrarouge entre une imprimante et un ordinateur, les deux appareils doivent disposer d’un émetteur et d’un récepteur. Une visibilité directe est nécessaire entre chaque couple émetteur-récepteur, avec une distance maximale de 3,7 m. La technologie infrarouge utilise une lumière invisible pour l’œil humain.

La technologie Bluetooth utilise une fréquence radio, non protégée par licence, pour une communication de courte portée ; elle est appréciée pour les casques sans fil et pour la synchronisation d’un assistant numérique personnel avec un ordinateur portable ou de bureau. Un adaptateur Bluetooth permet à un périphérique Bluetooth de se connecter à une imprimante, en général au moyen d’un port USB.

Wi-Fi est le nom répandu d’une technologie relativement nouvelle qui permet de connecter des ordinateurs à un réseau sans utiliser de câbles. Deux normes sont très utilisées pour la technologie Wi-Fi et elles commencent toutes deux par le numéro normalisé IEEE 802.11 :

  • La norme 802.11b transfère les données à un débit de 11 Mbits/s.
  • La norme 802.11g transfère les données à un débit de 54 Mbits/s. Les produits 802.11g sont rétrocompatibles avec la norme 802.11b.
