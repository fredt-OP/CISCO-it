1.1 Explication des certifications du secteur informatique  
  1.1.1 Identification de la formation et des certifications.  

Le terme « informatique » est un terme qui englobe le matériel, les logiciels, les réseaux et l’assistance technique proposés aux utilisateurs. IT Essentials: matériels et logiciels contient les informations nécessaires à un technicien pour qu’il réussisse dans l’informatique. Ce cours traite des sujets suivants :

  • ordinateurs personnels ;
  • procédures de travaux pratiques ;
  • dépannage ;
  • systèmes d’exploitation ;
  • ordinateurs portables ;
  • imprimantes et scanners ;
  • réseaux ;
  • sécurité ;
  • compétences en communication.

Le cours IT Essentials se centre sur les deux certifications basées sur les compétences en matière de matériel et de logiciels : CompTIA A+ et EUCIP. Ce cours constitue uniquement une introduction au monde de l’informatique. Un technicien a la possibilité d’étudier et d’obtenir les certifications suivantes :

  • CCNA – Cisco Certified Networking Associate ;
  • CCNP – Cisco Certified Networking Professional ;
  • CCIE – Cisco Certified Internetworking Expert ;
  • CISSP – Certified Information Systems Security Professional ;
  • MCP – Microsoft Certified Professional ;
  • MCSA – Microsoft Certified Systems Administrator ;
  • MCSE – Microsoft Certified Systems Engineer ;
  • Network+ – CompTIA Network Certification ;
  • Linux+ – CompTIA Linux Certification.

Les certifications informatiques peuvent être utilisées comme « crédits » pour les diplômes universitaires dans des domaines comme l’informatique et les télécommunications.
