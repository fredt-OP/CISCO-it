1.4 Identification des noms, objectifs et caractéristiques des composants internes  
  1.4.6 Identification des noms, objectifs et caractéristiques des disques de stockage  

Une unité de stockage lit ou écrit des informations sur un support de stockage magnétique ou optique. L’unité peut être utilisée pour stocker des données de façon permanente ou pour récupérer des informations à partir d’un disque multimédia. Les unités de stockage, le disque dur par exemple, peuvent être installées à l’intérieur du boîtier de l’ordinateur. En ce qui concerne la portabilité, certaines unités de stockage peuvent se connecter à l’ordinateur grâce à un port USB, un port FireWire ou un port SCSI. Ces unités de stockage portables sont parfois appelées disques amovibles et peuvent être utilisées sur plusieurs ordinateurs. Voici quelques types courants d’unités de stockage :

  • lecteur de disquettes ;
  • disque dur ;
  • lecteur optique ;
  • lecteur Flash ;
  • lecteur réseau.

Lecteur de disquette
Un lecteur de disquette est une unité de stockage utilisant des disquettes amovibles 3,5". Ces disquettes magnétiques peuvent stocker 720 Ko ou 1,44 Mo de données. Dans un ordinateur, le lecteur de disquette est généralement configuré comme le lecteur A:. Le lecteur de disquette peut être utilisé pour démarrer l’ordinateur, si ce dernier dispose d’une disquette amorçable. Un lecteur de disquette 5,25" est une technologie plus ancienne, rarement utilisée.

Disque dur
Un disque dur est une unité de stockage magnétique installée à l’intérieur de l’ordinateur. Le disque dur est utilisé comme stockage permanent des données. Dans un ordinateur, le disque dur est généralement configuré comme le lecteur C: et contient le système d’exploitation et les applications. Le disque dur est généralement configuré comme le premier lecteur dans la séquence de démarrage. La capacité de stockage d’un disque dur se mesure en milliards d’octets ou gigaoctets (Go). La vitesse d’un disque dur se mesure en tours par minute (tr/min). Plusieurs disques durs peuvent être ajoutés pour augmenter la capacité de stockage.

Lecteur optique
Un lecteur optique est une unité de stockage qui utilise des lasers pour lire des données sur le support optique. Il existe deux types de lecteurs optiques :

  • CD (Compact disc) ;
  • DVD (Digital versatile disc).

Les supports CD et DVD peuvent être préenregistrés (en lecture seule), enregistrables (une seule écriture) ou réenregistrables (plusieurs lectures et écritures). Les CD ont une capacité de stockage de données d’environ 700 Mo. Les DVD ont une capacité de stockage de données d’environ 8,5 Go sur un côté du disque.

Il existe plusieurs types de supports optiques :

  • CD-ROM : support de mémoire CD en lecture seule préenregistré.
  • CD-R : support CD enregistrable pouvant être enregistré une fois.
  • CD-RW : support CD réinscriptible pouvant être enregistré, effacé et réenregistré.
  • DVD-ROM  support de mémoire DVD en lecture seule préenregistré.
  • DVD-RAM : support de mémoire vive DVD pouvant être enregistré, effacé et réenregistré.
  • DVD+/-R : support DVD enregistrable pouvant être enregistré une fois.
  • DVD+/-RW : support DVD réinscriptible pouvant être enregistré, effacé et réenregistré.

Lecteur Flash
Un lecteur flash, également connu comme clé USB, est une unité de stockage amovible qui se connecte à un port USB. Un lecteur flash utilise un type spécial de mémoire ne requérant aucune alimentation pour conserver les données. Le système d’exploitation peut accéder à ces lecteurs de la même manière qu’il accède à d’autres types de lecteurs.

Types d’interfaces de lecteurs
Les disques durs et les lecteurs optiques sont fabriqués avec différentes interfaces, utilisées pour connecter le lecteur à l’ordinateur. Pour installer une unité de stockage sur un ordinateur, l’interface de connexion du lecteur doit être identique au contrôleur sur la carte mère. Voici certaines interfaces courantes de lecteurs :

  • IDE : l’interface IDE (Integrated Drive Electronics), également appelée interface ATA (Advanced Technology Attachment), est la première interface de contrôleur de lecteur qui connecte des ordinateurs et des disques durs. Une interface IDE utilise un connecteur à 40 broches.
  • EIDE : la technologie EIDE (Enhanced Integrated Drive Electronics), également appelée interface ATA-2, est une version mise à jour de l’interface de contrôleur de lecteur IDE. La technologie EIDE prend en charge des disques durs supérieurs à 512 Mo, permet un accès direct à la mémoire (vitesse) et utilise l’interface ATAPI (AT Attachment Packet Interface) pour accueillir les lecteurs optiques et les lecteurs de bande sur le bus EIDE. Une interface EIDE utilise un connecteur à 40 broches.
  • PATA  l’interface ATA parallèle se réfère à la version parallèle de l’interface de contrôleur de lecteur ATA.
  • SATA : l’interface ATA série se réfère à la version série de l’interface de contrôleur de lecteur ATA. Une interface SATA utilise un connecteur à 7 broches.
  • SCSI : l’interface SCSI (Small Computer System Interface) est une interface de contrôleur de lecteur pouvant connecter jusqu’à 15 lecteurs. SCSI permet de connecter des lecteurs internes et externes. Une interface SCSI utilise un connecteur à 50, 68 ou 80 broches.
