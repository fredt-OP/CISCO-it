1.2 Description d’un système informatique  

Un système informatique est composé de composants matériels et logiciels. Le matériel est l’équipement physique comme le boîtier, les disques de stockage, les claviers, les moniteurs, les câbles, les haut-parleurs et les imprimantes. Le terme « logiciel » inclut le système d’exploitation et les programmes. Le système d’exploitation indique à l’ordinateur comment fonctionner. Ces opérations peuvent inclure l’identification, l’accès et le traitement des informations. Les programmes ou les applications assurent différentes fonctions. Les programmes varient énormément, en fonction du type d’informations disponibles ou générées. Par exemple, les instructions d’établissement du solde d’un compte bancaire sont très différentes des instructions de simulation d’une réalité virtuelle sur Internet.
