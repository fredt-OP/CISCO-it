Wi-Fi

Marque initialement créée sous licence par l’association Wi-Fi Alliance afin de définir la technologie intégrée aux réseaux sans fil. Le Wi-Fi est basé sur les normes IEEE 802.11.

WTLS (Wireless Transport Layer Security)

Couche apportant une fonction de sécurité pour les périphériques mobiles utilisant le protocole WAP (Wireless Applications Protocol).
