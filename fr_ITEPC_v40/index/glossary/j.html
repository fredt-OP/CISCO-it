Java

Langage de programmation permettant d’exécuter les applets dans un navigateur Web. La calculatrice et le compteur sont des exemples d’applets.

JavaScript

Langage de programmation développé pour interagir avec le code source HTML des sites Web interactifs. La bannière rotative et la fenêtre contextuelle sont des exemples d’applications JavaScript.

jeu de commandes compatibles Hayes

Ensemble de commandes AT que la plupart des logiciels de modem utilisent. Ce jeu de commandes porte le nom de la société Hayes Microcomputer Products Company, qui a été la première à les définir.

jeu de composants

Puces situées sur la carte mère, permettant à l’unité centrale de communiquer et d’interagir avec les autres composants de l’ordinateur.
