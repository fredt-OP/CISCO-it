NetBIOS (Network Basic Input/Output System)

Interface de programmation d’application (API) utilisée par les applications d’un réseau local IBM pour effectuer des requêtes de services auprès des processus réseau de couche inférieure. Ces services peuvent inclure l’établissement et l’arrêt d’une session, ainsi que le transfert de données.

nettoyage de disque

Logiciel de gestion de disque utilisé pour libérer de l’espace sur le disque dur. Il recherche les fichiers pouvant être supprimés en toute sécurité, tels que les fichiers temporaires Internet.

NetView

Architecture de gestion de réseaux IBM et applications connexes. NetView est une application VTAM (méthode d’accès pour les télécommunications en mode virtuel) utilisée pour la gestion des ordinateurs centraux des réseaux SNA (Systems Network Architecture).

noeud

1) Point terminal d’une connexion réseau ou jonction entre plusieurs lignes d’un réseau. Les noeuds peuvent être des processeurs, des contrôleurs ou des stations de travail. Leurs différentes fonctions, notamment la fonctionnalité de routage, sont variables. Ils peuvent être interconnectés et servir de points de contrôle dans un réseau. Le terme « noeud » désigne parfois, de manière générique, toute entité pouvant accéder à un réseau. Il est souvent utilisé, de façon interchangeable, pour désigner un périphérique.
2) Dans l’architecture SNA, le noeud est le composant de base d’un réseau et le point auquel une ou plusieurs unités fonctionnelles sont reliées à des canaux ou des circuits de données.

NSLOOKUP

Commande qui renvoie l’adresse IP d’un hôte sur base de son nom. Cette commande peut également effectuer l’action inverse et retrouver le nom d’un hôte à partir de son adresse IP.

NTDETECT

Programme utilisé par les systèmes Intel pour détecter le matériel installé.

NTFS (New Technology File System)

Type de système de fichiers qui offre une meilleure tolérance aux pannes par rapport aux systèmes de fichiers traditionnels, ainsi qu’une sécurité au niveau des fichiers.
