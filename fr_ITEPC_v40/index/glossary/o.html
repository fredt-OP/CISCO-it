Observateur d’événements

Application chargée de contrôler les événements système, les événements d’application et les événements de sécurité.

octet

Unité de mesure utilisée pour décrire la taille d’un fichier de données, le volume disponible sur un disque ou autre support de stockage ou le volume de données envoyé via un réseau. Un octet est constitué de 8 bits de données.

Ohm's Law

Relation mathématique entre le courant, la résistance et la tension, où la tension est égale au courant multiplié par la résistance.

options de gestion du support

Options grâce auxquelles une imprimante gère le support, notamment l’orientation, la taille et le grammage du papier.

options de sortie de l’imprimante

Ces options indiquent comment l’encre ou le toner est appliquée sur le papier et incluent la gestion des couleurs, la qualité et la vitesse d’impression.

ordinateur

Appareil électrique capable d’exécuter une liste d’instructions et d’effectuer des calculs sur base de ces instructions.

ordinateur de bureau

Type d’ordinateur conçu pour être utilisé sur un bureau, le moniteur étant généralement placé sur l’unité afin de réduire l’encombrement. Les ordinateurs de bureau ne sont pas mobiles, contrairement aux ordinateurs portables.

ordinateur portable

Ordinateur, de format compact, spécialement conçu pour une utilisation mobile, mais offrant les mêmes fonctionnalités qu’un ordinateur de bureau. Les équipements matériels d’ordinateur portable sont brevetés et plus chers que ceux d’un ordinateur de bureau.

outil de configuration

Outil de gestion des services ou outil de service de gestion des éléments, doté d’une interface graphique utilisateur.

outils de diagnostic

Utilitaires permettant de contrôler le serveur d’un réseau.
