15.4 Implémentation du réseau client 
  15.4.2 Configuration des ressources réseau et Internet du client  

Une fois le réseau installé et testé, vous devez configurer un navigateur Web, par exemple Microsoft Internet Explorer (IE). Vous pouvez configurer les paramètres du navigateur et effectuer des tâches de maintenance dans la boîte de dialogue Propriétés Internet, présentée dans la Figure 1.

Fichiers Internet temporaires
Lorsqu’un système d’exploitation tel que Windows XP est installé, Internet Explorer est également installé par défaut. Avec IE, chaque fois que vous visitez un site Web, de nombreux fichiers sont téléchargés sur votre ordinateur dans le dossier Fichiers Internet temporaires. La plupart de ces fichiers sont des images représentant des bannières et autres composants d’un site Web.

Les fichiers Internet temporaires sont stockés sur votre ordinateur, afin que le navigateur puisse charger leur contenu plus rapidement la prochaine fois que vous visitez un site Web déjà visité auparavant. En fonction du nombre de sites Web que vous visitez, le dossier Fichiers Internet temporaires peut se remplir rapidement. Il est recommandé, bien que cette tâche ne soit pas urgente, de supprimer occasionnellement ces fichiers. Cette opération est particulièrement importante lorsque vous avez consulté votre banque en ligne ou saisi des informations personnelles dans votre navigateur Web.

Navigateur par défaut
Vous pouvez choisir quel navigateur Windows doit utiliser par défaut. Sélectionnez Démarrer > Exécuter, saisissez l’adresse d’un site Web, puis cliquez sur OK. Le site Web s’ouvre dans le navigateur actuellement défini comme navigateur par défaut.

Si vous souhaitez qu’Internet Explorer soit votre navigateur par défaut, ouvrez-le. Dans la barre d’outils, sélectionnez Outils > Options Internet…. Dans l’onglet Programmes, vous pouvez vérifier si IE est votre navigateur par défaut et le sélectionner si vous le souhaitez.

Partage des fichiers
Les utilisateurs peuvent partager des ressources via le réseau. Il est ainsi possible de partager un fichier unique, des dossiers spécifiques ou un disque entier, comme indiqué dans la Figure 2.

Pour partager un fichier, vous devez tout d’abord le copier dans un dossier. Cliquez avec le bouton droit de la souris sur le dossier et sélectionnez Partage et sécurité. Sélectionnez ensuite Partager ce dossier. Vous pouvez identifier les utilisateurs ayant accès au dossier et les autorisations dont ils disposent vis-à-vis des objets qu’il contient. La Figure 3 représente la fenêtre d’autorisations d’un dossier partagé.

Les autorisations définissent le type d’accès d’un utilisateur pour un fichier ou un dossier donné :

  • Lecture : permet à l’utilisateur d’afficher le nom des fichiers et des sous-dossiers, de naviguer parmi les sous-dossiers, d’afficher les données contenues dans les fichiers et d’exécuter les fichiers programmes.
  • Modification : octroie les mêmes autorisations que le niveau Lecture, mais permet en plus à l’utilisateur d’ajouter des fichiers et des sous-dossiers, de modifier les données contenues dans les fichiers et de supprimer des sous-dossiers et des fichiers.
  • Contrôle complet : octroie toutes les autorisations des niveaux Modification et Lecture. Si le fichier ou le dossier se situe dans une partition NTFS, le niveau Contrôle complet vous permet de modifier les autorisations du fichier ou du dossier et d’en devenir propriétaire.

Windows XP Professionnel limite à 10 le nombre de connexions simultanées en partage de fichiers.

Partage d’imprimante
Pour partager une imprimante, sélectionnez Démarrer > Panneau de configuration > Imprimantes et télécopieurs. Cliquez avec le bouton droit de la souris sur l’icône de l’imprimante, puis sélectionnez Partager. Cliquez sur Partager cette imprimante, puis sur OK. L’imprimante est désormais disponible pour les autres ordinateurs du réseau.

Pour accéder à une imprimante partagée, sélectionnez Démarrer > Panneau de configuration > Imprimantes et télécopieurs. Cliquez sur Fichier > Ajouter une imprimante. Servez-vous de l’assistant Ajout d’imprimante pour rechercher et installer l’imprimante partagée souhaitée.
