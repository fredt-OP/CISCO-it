  15.5.2 Installation et configuration des routeurs sans fil  

Lors de l’installation d’un réseau sans fil, vous devez décider où placer les points d’accès, puis les configurer. Les étapes suivantes décrivent l’installation d’un point d’accès :

  1. Servez-vous du plan d’étage pour trouver l’emplacement idéal des points d’accès, qui offrira une couverture optimale. Le meilleur emplacement pour un point d’accès sans fil est au centre de la zone couverte, avec les différents périphériques sans fil en visuel.
  2. Connectez le point d’accès au réseau existant. Le routeur Linksys WRT300N est équipé de cinq ports, situés à l’arrière. Branchez un modem câble ou DSL sur le port marqué « Internet ». La logique de commutation du périphérique transfère tous les paquets via ce port, lors d’une communication entre Internet et les ordinateurs connectés. Branchez un ordinateur sur l’un des ports restants, afin d’accéder aux pages Web de configuration.
  3. Mettez le modem large bande sous tension et branchez le cordon d’alimentation sur le routeur. Lorsque le modem a établi une connexion avec le FAI, le routeur communique automatiquement avec le modem pour recevoir les informations réseau en provenance du FAI, nécessaires pour accéder à Internet : adresse IP, masque de sous-réseau et adresses de serveur DNS.
  4. Une fois que le routeur a établi une communication avec le modem, vous devez le configurer afin qu’il communique avec les périphériques du réseau. Mettez en marche l’ordinateur connecté au routeur. Ouvrez un navigateur Web. Dans le champ Adresse, saisissez 192.168.1.1. Il s’agit de l’adresse par défaut pour la configuration et la gestion du routeur.
  5. Une fenêtre de sécurité vous invite à vous authentifier pour pouvoir accéder aux écrans de configuration du routeur. Le champ du nom d’utilisateur doit rester vide. Saisissez admin comme mot de passe (par défaut). Une fois que vous êtes connecté, le premier écran de configuration s’affiche.
  6. Procédez à la configuration. L’écran de configuration contient plusieurs onglets, eux-mêmes possédant des sous-onglets. Vous devez cliquer sur Enregistrer les paramètres, en bas de chaque écran, après avoir effectué vos modifications.

Lorsque vous utilisez les écrans de configuration d’un routeur 300N, vous pouvez cliquer sur l’onglet d’aide pour obtenir des informations sur un onglet spécifique. Pour obtenir des informations plus détaillées que celles figurant dans l’aide, consultez le manuel d’utilisation du système.
