15.5 Mise à niveau du réseau client  
  15.5.1 Installation et configuration d’une carte réseau sans fil  

Pour pouvoir se connecter à un réseau sans fil, votre ordinateur doit posséder une interface de réseau sans fil. Cette interface sert à communiquer avec les autres périphériques sans fil, par exemple les ordinateurs, les imprimantes ou les points d’accès sans fil.

Avant d’acheter une carte réseau sans fil, vous devez vous assurer qu’elle est compatible avec les autres équipements sans fil déjà installés sur le réseau. En outre, vous devez vérifier que la carte est au format approprié et s’insère dans un ordinateur de bureau ou un ordinateur portable. Une carte USB sans fil peut être utilisée sur n’importe quel ordinateur, de bureau ou portable, disposant d’un port USB ouvert.

Pour installer une carte réseau sans fil sur un ordinateur de bureau, vous devez retirer le capot du boîtier. Insérez ensuite la carte dans un logement PCI ou PCI Express disponible. Certaines cartes réseau sans fil sont équipées d’une antenne, placée à l’arrière de la carte. Certaines antennes sont branchées à l’aide d’un câble, ce qui permet de les déplacer et qu’aucun obstacle ou objet ne gêne l’émission, et donc la connexion.

Une fois la carte sans fil installée, observez les étapes de configuration requises. Celles-ci incluent la configuration des pilotes du périphérique et la saisie des informations d’adresse réseau. Une fois la configuration terminée, l’ordinateur doit être capable de détecter le réseau local sans fil et de s’y connecter.

Les cartes réseau sans fil font parfois appel à un assistant pour la connexion au réseau. Dans ce cas, il vous suffit d’insérer dans le lecteur le CD fourni avec la carte et d’observer les instructions qui s’affichent pour vous connecter.
