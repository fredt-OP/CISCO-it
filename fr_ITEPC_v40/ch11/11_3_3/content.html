11.3 Description des situations nécessitant le remplacement de composants d’un ordinateur  
  11.3.3 Sélection d’un processeur et d’un ensemble dissipateur thermique/ventilateur  

Remplacez le processeur lorsqu’il est défaillant ou qu’il n’est plus adapté aux applications actuelles. Par exemple, un client peut avoir acheté une application graphique avancée, comme illustré dans la Figure 1. L’application peut mal fonctionner car elle nécessite un processeur plus rapide que le processeur actuel.

Avant d’acheter un processeur, assurez-vous qu’il est compatible avec la carte mère existante :

  • le nouveau processeur doit utiliser le même type d’interface de connexion et le même jeu de composants ;
  • le BIOS doit prendre en charge le nouveau processeur ;
  • le nouveau processeur peut nécessiter un ensemble dissipateur thermique/ventilateur différent.

Les sites Web des fabricants constituent une bonne ressource pour se renseigner sur la compatibilité entre les processeurs et d’autres éléments. Lorsque vous mettez à niveau le processeur, assurez-vous que la tension est correcte. Un module régulateur de tension (VRM) est intégré dans la carte mère. La tension du processeur peut être réglée avec des cavaliers ou des commutateurs situés sur la carte mère. La Figure 2 montre deux processeurs similaires utilisant différentes interfaces de connexion.
