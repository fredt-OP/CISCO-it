11.3 Description des situations nécessitant le remplacement de composants d’un ordinateur  
  11.3.7 Sélection des périphériques d’entrée et de sortie  

Un périphérique d’entrée peut être tout élément matériel transférant des informations à un ordinateur :

  • souris ;
  • clavier ;
  • scanner ;
  • appareil photo ;
  • capteur de commande de processus ;
  • interface MIDI ;
  • microphone.

Un périphérique de sortie transfère des informations vers l’extérieur de l’ordinateur :

  • écran ;
  • projecteur ;
  • imprimante ;
  • équipement de commande de processus ;
  • haut-parleur.

Pour sélectionner des périphériques d’entrée et de sortie, vous devez d’abord savoir ce que souhaite le client. Ensuite, vous devez sélectionner le matériel et les logiciels en recherchant de possibles solutions sur Internet. Après avoir défini le périphérique d’entrée ou de sortie souhaité par le client, vous devez déterminer la manière dont il doit être connecté à l’ordinateur. La Figure 1 présente des symboles de port d’entrée et de sortie courants.

Les techniciens doivent avoir une bonne connaissance des différents types d’interface suivants :

  • USB 1.1 : transfère les données avec un débit maximal de 12 Mbits/s ;
  • USB 2,0 : transfère les données avec un débit maximal de 480 Mbits/s ;
  • IEEE 1394 (FireWire) : transfère les données avec un débit de 100, 200 ou 400 Mbits/s ;
  • Parallèle (IEEE 1284)  : transfère les données avec un débit maximal de 3 Mbits/s ;
  • Série (RS-232) : les versions antérieures se limitaient à un taux de transfert de 20 Kbits/s ; les nouvelles versions peuvent atteindre 1,5 Mbits/s ;
  • SCSI (Ultra-320 SCSI) : connecte jusqu’à 15 périphériques avec un taux de transfert de 320 Mbits/s.
