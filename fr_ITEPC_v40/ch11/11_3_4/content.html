11.3 Description des situations nécessitant le remplacement de composants d’un ordinateur  
  11.3.4 Sélection de la mémoire vive  

Lorsqu’une application se bloque ou que l’ordinateur affiche des messages d’erreur fréquents, il faut peut-être rajouter de la mémoire vive. Pour savoir si le problème vient de la mémoire vive, remplacez l’ancien module de mémoire vive, comme indiqué dans la Figure 1. Redémarrez l’ordinateur pour voir si l’application fonctionne correctement.

REMARQUE : pour fermer une application bloquée, appuyez sur Ctrl+Alt+Suppr afin d’ouvrir le Gestionnaire des tâches. Dans la fenêtre, sélectionnez l’application concernée. Cliquez sur Fin de tâche, comme indiqué dans la Figure 2.

Lorsque vous choisissez une nouvelle mémoire vive, vous devez vous assurer qu’elle est compatible avec la carte mère actuelle. Par ailleurs, la mémoire vive doit être du même type que celle installée sur l’ordinateur. La vitesse de la nouvelle mémoire vive doit être égale ou supérieure à celle de la mémoire vive existante. Pour vous aider, lorsque vous allez acheter la mémoire vive de remplacement, prenez avec vous le module de mémoire d’origine.
