  5.5.4 Installation, désinstallation et accès à une application  

En tant que technicien, vous devrez ajouter et supprimer des logiciels sur les ordinateurs de vos clients. La plupart des applications utilisent un processus d’installation automatique lorsqu’un CD est inséré dans le lecteur optique. Le processus d’installation met à jour l’utilitaire Ajout/Suppression de programmes. L’utilisateur doit cliquer sur certaines options dans l’assistant d’installation et fournir des informations lorsqu’il y est invité.

Applet Ajout/Suppression de programmes
Microsoft recommande de toujours utiliser l’utilitaire Ajout/Suppression de programmes, comme indiqué dans la Figure 1, lors de l’installation ou de la suppression d’applications. Lorsque vous utilisez l’utilitaire Ajout/Suppression de programmes pour installer une application, celui-ci effectue un suivi des fichiers d’installation, afin que l’application puisse être désinstallée complètement, si vous le souhaitez. Pour ouvrir l’applet Ajout/Suppression de programmes, suivez le chemin :

Démarrer > Panneau de configuration > Ajout/Suppression de programmes

Ajout d’une application
Si un programme ou une application ne s’installe pas automatiquement lorsque le CD est inséré, vous pouvez utiliser l’applet Ajout/Suppression de programmes pour installer l’application, comme indiqué dans la Figure 2. Cliquez sur le bouton Ajouter de nouveaux programmes et sélectionnez l’emplacement où se trouve l’application. Windows installe l’application pour vous.

Une fois l’application installée, vous pouvez la démarrer à partir du menu Démarrer ou d’une icône de raccourci installée par l’application sur le Bureau. Vérifiez que l’application fonctionne correctement. Si l’application rencontre des problèmes, corrigez-les ou désinstallez l’application. Certaines applications, comme Microsoft Office, proposent une option de réparation dans le processus d’installation. Vous pouvez l’utiliser pour tenter de corriger un programme qui ne fonctionne pas correctement.

Désinstallation d’une application
Si une application n’est pas correctement désinstallée, vous risquez de laisser des fichiers sur le disque dur et des paramètres inutiles dans le registre. Bien que n’entraînant généralement pas de problèmes, cela diminue l’espace libre sur le disque dur, les ressources système et la vitesse de lecture du registre. La Figure 3 présente l’applet Ajout/Suppression de programmes, à utiliser pour désinstaller des programmes que vous n’utilisez plus. L’assistant vous guide au cours du processus de suppression du logiciel et supprime tous les fichiers installés.
