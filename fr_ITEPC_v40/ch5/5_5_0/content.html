5.5 Navigation dans une interface graphique utilisateur (Windows)  

Le système d’exploitation fournit une interface utilisateur qui vous permet d’interagir avec l’ordinateur. Pour naviguer dans le système de fichiers et exécuter les applications d’un système d’exploitation, vous avez le choix entre deux méthodes :

  • Une interface graphique utilisateur fournit des représentations graphiques (icônes) de tous les fichiers, dossiers et programmes présents sur un ordinateur. Vous manipulez ces icônes à l’aide d’un pointeur contrôlé par une souris ou un périphérique similaire. Le pointeur vous permet de déplacer les icônes, en les faisant glisser et en les positionnant, ainsi que d’exécuter les programmes en cliquant dessus.
  • Une interface de ligne de commande (ILC) est de type texte. Vous devez saisir des commandes pour manipuler les fichiers et exécuter les programmes.

À la fin de cette section, vous parviendrez à :

  • manipuler les éléments du bureau ;
  • explorer les applets du Panneau de configuration ;
  • explorer les outils d’administration ;
  • installer, désinstaller et accéder à une application ;
  • décrire la mise à niveau des systèmes d’exploitation.
