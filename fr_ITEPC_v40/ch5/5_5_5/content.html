  5.5.5 Description de la mise à niveau d’un système d’exploitation  

Il peut parfois être nécessaire de mettre à niveau un système d’exploitation. Avant de le faire, vérifiez la configuration minimale du nouveau système d’exploitation, pour vous assurer que l’ordinateur répond aux spécifications minimales requises. Consultez la liste de compatibilité matérielle pour vérifier que le matériel est compatible avec le nouveau système d’exploitation. Sauvegardez toutes les données avant de mettre à niveau le système d’exploitation, au cas où l’installation rencontrerait un problème.

Il est plus rapide de mettre à niveau un système informatique de Windows 2000 vers Windows XP que d’installer Windows XP. L’utilitaire de configuration Windows XP remplace les fichiers Windows 2000 existants par les fichiers Windows XP lors du processus de mise à niveau. Toutefois, les applications et paramètres existants sont sauvegardés.

Mise à niveau du système d’exploitation

  1. Insérez le CD Windows XP dans le lecteur CD-ROM pour lancer le processus de mise à niveau. Sélectionnez Démarrer > Exécuter.
  2. Dans la boîte de dialogue Exécuter, tapez D:\i386\winnt32, où D est la lettre du lecteur, puis appuyez sur Entrée. L’Assistant Installation de Windows XP s’affiche.
  3. Sélectionnez Mettre à niveau vers Windows XP, puis cliquez sur Suivant. La page Contrat de licence s’affiche.
  4. Lisez le contrat de licence et cliquez sur le bouton pour en accepter les conditions.
  5. Cliquez sur Suivant. La page Mise à niveau vers le système de fichiers NTFS de Windows XP s’affiche.
  6. Suivez les invites et effectuez la mise à niveau. Une fois l’installation terminée, l’ordinateur redémarre.

REMARQUE : l’Assistant Installation de Windows XP peut démarrer automatiquement lorsque le CD est inséré dans le lecteur optique.
