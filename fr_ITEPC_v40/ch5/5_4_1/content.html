5.4 Installation d’un système d’exploitation  
  5.4.1 Identification des procédures de configuration d’un disque dur  

L’installation d’un système d’exploitation inclut son installation et son amorçage initial. Bien qu’il soit possible d’installer un système d’exploitation via un réseau à partir d’un serveur ou d’un disque dur local, la méthode d’installation la plus courante utilise les CD et les DVD. Pour installer un système d’exploitation avec un CD ou un DVD, configurez d’abord le BIOS pour amorcer le système à partir du CD ou du DVD.

Partition et formatage
Avant d’installer un système d’exploitation sur un disque dur, celui-ci doit être partitionné et formaté. Lorsqu’un disque dur est partitionné, il est divisé logiquement en une ou plusieurs zones. Lorsqu’un disque dur est formaté, les partitions sont préparées pour contenir des fichiers et applications. Pendant la phase d’installation, la plupart des systèmes d’exploitation partionnent et formatent automatiquement le disque dur. Un technicien doit comprendre la procédure de configuration d’un disque dur. Les termes suivants se réfèrent à la configuration d’un disque dur :

  • Partition principale : il s’agit généralement de la première partition. Une partition principale ne peut pas être subdivisée en sections plus petites. Un disque dur peut compter jusqu’à quatre partitions.
  • Partition active : cette partition est utilisée par le système d’exploitation pour amorcer l’ordinateur. Seule une partition principale peut être marquée comme active.
  • Partition étendue : cette partition utilise normalement l’espace libre restant sur un disque dur ou elle se substitue à une partition principale. Il ne peut y avoir qu’une partition étendue par disque dur, laquelle peut être subdivisée en sections plus petites appelées unités logiques.
  • Unité logique : ce lecteur est une section d’une partition étendue pouvant être utilisée pour séparer des informations à des fins administratives.
  • Formatage : ce processus prépare un système de fichiers dans une partition pour le stockage des fichiers.
  • Cluster : un cluster est également appelé unité d’allocation de fichiers. C’est la plus petite unité d’espace utilisée pour le stockage des données.
  • Piste : une piste est un cercle complet de données sur un côté d’un plateau de disque dur. Une piste est divisée en groupes de 512 octets appelés secteurs.
  • Cylindre : un cylindre est un ensemble de pistes alignées les unes au-dessus des autres pour former un cylindre.
  • Mappage de lecteur : le mappage de lecteur est une lettre assignée à une unité physique ou logique.
