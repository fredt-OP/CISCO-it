5.4 Installation d’un système d’exploitation  
  5.4.5 Fin de l’installation  

Après la copie sur le disque dur, lors de l’installation Windows, de tous les fichiers du système d’exploitation nécessaires, l’ordinateur se réamorce et vous invite à ouvrir une session pour la première fois.

Vous devez enregistrer Windows XP. Comme indiqué dans la Figure 1, vous devez également effectuer la vérification destinée à garantir que vous utilisez une copie légale du système d’exploitation. Cela vous permet de télécharger des correctifs et des service packs. Pour ce faire, vous devez vous connecter à Internet.

En fonction de l’ancienneté du support au moment de l’installation, il sera peut-être nécessaire d’installer des mises à jour. Comme indiqué dans la Figure 2, vous pouvez utiliser la fonction Windows Update en cliquant sur le menu Démarrer, pour rechercher de nouveaux logiciels et accomplir les actions suivantes :

  • installer tous les service packs ;
  • installer tous les correctifs.

Démarrer > Programmes > Accessoires > Outils système > Windows Update

Vous devez également vérifier que tout le matériel est installé correctement. Comme indiqué dans la Figure 3, vous pouvez utiliser le Gestionnaire de périphériques pour localiser les problèmes et installer les bons pilotes ou les pilotes mis à jour, en suivant le chemin suivant :

Démarrer > Panneau de configuration > Système > Matériel > Gestionnaire de périphériques

Dans le Gestionnaire de périphériques, des icônes d’avertissement sont symbolisées par un point d’exclamation jaune ou un « X » rouge. Le point d’exclamation jaune signifie qu’il y a un problème avec le périphérique. Pour afficher un descriptif du problème, cliquez avec le bouton droit de la souris sur le périphérique et sélectionnez Propriétés. Le « X » rouge signifie qu’un périphérique a été désactivé. Pour activer un périphérique, cliquez dessus avec le bouton droit de la souris et sélectionnez Activer. Pour ouvrir une catégorie qui n’est pas encore développée, cliquez sur le signe (+).

REMARQUE : lorsque Windows détecte une erreur système, le service Windows error reporting (WER) de Microsoft affiche une boîte de dialogue. Si vous choisissez d’envoyer le rapport, WER collecte des informations sur l’application et le module impliqués dans l’erreur et envoie ces informations à Microsoft.
