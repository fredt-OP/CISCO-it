5.4 Installation d’un système d’exploitation  
  5.4.3 Installation du système d’exploitation avec les paramètres par défaut  

Lorsque vous installez Windows XP, l’assistant d’installation vous donne le choix entre l’installation des paramètres standard (par défaut) et l’installation des paramètres personnalisés. L’utilisation des paramètres standard augmente la probabilité de réussite de l’installation. Toutefois, l’utilisateur doit fournir les informations suivantes pendant l’installation :

  • normes et formats définissant la monnaie et les nombres ;
  • langage de saisie du texte ;
  • nom de l’utilisateur et de l’entreprise ;
  • clé du produit ;
  • nom de l’ordinateur ;
  • mot de passe administrateur ;
  • paramètres de date et d’heure ;
  • paramètres réseau ;
  • informations de domaine ou de groupe de travail.

Lorsqu’un ordinateur est amorcé avec le CD d’installation Windows, l’installation de Windows XP démarre, avec trois options :

  • Installer XP : appuyez sur la touche Entrée pour lancer l’installation et installer le système d’exploitation XP.
  • Réparer XP : appuyez sur la touche R pour ouvrir la console de récupération et réparer une installation.
  • Quitter : appuyez sur la touche F3 pour quitter sans installer Windows XP.

Pour cette section, sélectionnez l’option Installer XP.
