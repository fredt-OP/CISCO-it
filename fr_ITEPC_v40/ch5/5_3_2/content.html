5.3 Détermination du système d’exploitation en fonction des besoins des clients  
  5.3.2 Détermination de la configuration matérielle minimale et de la compatibilité avec la plateforme du système d’exploitation  

Les systèmes d’exploitation doivent avoir une configuration matérielle minimale pour que le système d’exploitation puisse être installé et fonctionner correctement. La Figure 1 est un tableau de la configuration matérielle minimale requise pour les systèmes d’exploitation Windows 2000, Windows XP Professionnel et Windows XP Édition familiale.

Identifiez l’équipement de votre client. Si des mises à niveau matérielles sont nécessaires pour atteindre la configuration minimale requise et pouvoir installer un système d’exploitation, procédez à une analyse des coûts pour déterminer la meilleure action à entreprendre. Dans certains cas, il peut être moins coûteux pour le client d’acheter un nouvel ordinateur que de mettre à niveau son système. Dans d’autres cas, il peut être plus économique de mettre à niveau un ou plusieurs composants suivants :

  • mémoire vive ;
  • disque dur ;
  • unité centrale ;
  • adaptateur vidéo.

REMARQUE : parfois, la configuration des applications peut dépasser la configuration matérielle du système d’exploitation. Pour que l’application fonctionne correctement, il est alors nécessaire de mettre à niveau la configuration matérielle.

Une fois que vous avez défini la configuration matérielle minimale nécessaire à un système d’exploitation, vous devez vous assurer que tout le matériel présent dans l’ordinateur est compatible avec le système d’exploitation que vous avez sélectionné pour votre client.

Liste de compatibilité matérielle
La plupart des systèmes d’exploitation s’accompagnent d’une liste de compatibilité matérielle, que vous pouvez trouver sur le site Web du fabricant, comme indiqué sur la Figure 2. Ces listes répertorient en détail le matériel testé et déclaré comme étant compatible avec le système d’exploitation. Si certains composants matériels du client ne figurent pas dans la liste, ils ont peut-être besoin d’être mis à niveau, afin de correspondre à ceux de la liste.

REMARQUE : une liste de compatibilité matérielle n’est pas forcément tenue à jour en permanence, auquel cas elle n’est pas une référence suffisante.
