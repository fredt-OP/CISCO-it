5.4 Installation d’un système d’exploitation  
  5.4.6 Description des options d’installation personnalisées  

L’installation d’un système d’exploitation sur un seul ordinateur prend du temps. Imaginez le temps que cela prendrait pour installer des systèmes d’exploitation un par un sur plusieurs ordinateurs dans une grande entreprise. Pour vous simplifier la tâche, vous pouvez utiliser l’outil System Preparation (Sysprep) de Microsoft, qui vous permet d’installer et de configurer le même système d’exploitation sur plusieurs ordinateurs. Sysprep prépare un système d’exploitation destiné à être utilisé sur des ordinateurs aux configurations matérielles différentes. Avec Sysprep et une application de clonage de disque, les techniciens peuvent installer rapidement un système d’exploitation, effectuer les dernières étapes de la configuration du système d’exploitation et installer des applications.

Clonage de disque
Le clonage de disque crée une image d’un disque dur dans un ordinateur. Pour cloner un disque dur, procédez comme suit :

  1. Créez une installation maître sur un ordinateur. Celle-ci comprend le système d’exploitation, les applications et les paramètres de configuration qui seront utilisés par les autres ordinateurs de l’organisation.
  2. Exécutez Sysprep.
  3. Créez une image du disque de l’ordinateur configuré à l’aide d’un programme tiers de clonage de disque.
  4. Copiez l’image du disque sur un serveur. Lorsque l’ordinateur de destination est amorcé, une version réduite du programme d’installation de Windows se lance. Le programme d’installation crée alors un nouvel ID de sécurité, installe les pilotes du matériel, crée des comptes utilisateur et configure les paramètres réseau pour terminer l’installation du système d’exploitation.
