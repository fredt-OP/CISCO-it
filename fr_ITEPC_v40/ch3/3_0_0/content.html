3.0 Introduction  

L’assemblage d’ordinateurs représente une grande partie du travail d’un technicien. En tant que technicien, vous devrez travailler avec les composants d’un ordinateur de manière logique et méthodique. Comme dans tous les métiers, les compétences en matière d’assemblage d’ordinateurs augmentent considérablement avec la pratique.

À la fin de ce chapitre, vous parviendrez à :

  • ouvrir le boîtier ;
  • installer l’alimentation ;
  • relier les composants à la carte mère et installer la carte mère ;
  • installer les lecteurs internes ;
  • installer les lecteurs dans les compartiments externes ;
  • installer les adaptateurs ;
  • connecter tous les câbles internes ;
  • replacer les panneaux latéraux et connecter les câbles externes à l’ordinateur ;
  • amorcer l’ordinateur pour la première fois.
