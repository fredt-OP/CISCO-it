3.9 Premier amorçage de l’ordinateur  
  3.9.2 Description de la configuration du BIOS  

Le BIOS contient un programme de configuration utilisé pour configurer les paramètres des périphériques. Les données de configuration sont enregistrées sur une puce mémoire spéciale appelée CMOS (complementary metal-oxide semiconductor), comme indiqué dans la Figure 1. CMOS est alimenté par la batterie se trouvant dans l’ordinateur. Si la batterie ne fonctionne plus, toutes les données de configuration du BIOS sont perdues. Si tel est le cas, remplacez la batterie et configurez à nouveau les paramètres du BIOS.

Pour entrer dans le programme de configuration du BIOS, vous devez appuyer sur la touche ou la séquence de touches appropriée lors du POST. La plupart des ordinateurs utilisent la touche Suppr. Votre ordinateur peut utiliser une autre touche ou combinaison de touches.

La Figure 2 présente un exemple de programme de configuration du BIOS. Voici quelques options du menu de configuration du BIOS courantes :

  • Main : heure système, date, type de disque dur, etc.
  • Advanced : paramètres de port infrarouge, paramètres de port parallèle, etc.
  • Security : paramètres de mot de passe pour l’utilitaire de configuration
  • Others : alarme batterie faible, bip système, etc.
  • Boot : ordre d’amorçage de l’ordinateur
  • Exit : quitter l’utilitaire de configuration
