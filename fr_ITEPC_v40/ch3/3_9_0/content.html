3.9 Premier amorçage de l’ordinateur  

Lorsque l’ordinateur est amorcé, le BIOS (basic input/output system) vérifie tous les composants internes. Cette vérification est appelée test automatique de mise sous tension (Power-On Self Test, POST).

À la fin de cette section, vous parviendrez à :

  • identifier les codes de bips ;
  • décrire la configuration du BIOS.
