3.10 Résumé  

Ce chapitre a détaillé les étapes à suivre pour assembler un ordinateur et amorcer le système pour la première fois. Voici quelques points importants à retenir :

  • les boîtiers d’ordinateur ont des tailles et configurations diverses ; de nombreux composants de l’ordinateur doivent correspondre au format du boîtier ;
  • le processeur est installé sur la carte mère avec un ensemble dissipateur thermique/ventilateur ;
  • la mémoire vive est installée dans les emplacements de mémoire vive situés sur la carte mère ;
  • les adaptateurs sont installés dans les logements d’extension PCI et PCIe situés sur la carte mère ;
  • les disques durs sont installés dans des compartiments de lecteurs 3,5", à l’intérieur du boîtier ;
  • les disques optiques sont installés dans des compartiments de lecteurs 5,25", auxquels on peut accéder par l’extérieur du boîtier ;
  • les lecteurs de disquette sont installés dans des compartiments de lecteurs 3,5", auxquels on peut accéder par l’extérieur du boîtier ;
  • les câbles d’alimentation sont connectés à tous les lecteurs et à la carte mère ;
  • les câbles de données internes transfèrent les données à tous les lecteurs ;
  • les câbles externes relient les périphériques à l’ordinateur ;
  • les codes de bips indiquent des dysfonctionnements du matériel ;
  • le programme de configuration du BIOS est utilisé pour afficher des informations sur les composants de l’ordinateur et il permet à l’utilisateur de modifier les paramètres système.
