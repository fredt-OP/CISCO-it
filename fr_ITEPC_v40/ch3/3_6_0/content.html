3.6 Installation des adaptateurs  

Les adaptateurs ajoutent des fonctionnalités à un ordinateur. Les adaptateurs doivent être compatibles avec le logement d’extension. Cette section porte sur l’installation de trois types d’adaptateurs :

  • carte réseau PCIe x1 ;
  • carte réseau sans fil PCI ;
  • adaptateur vidéo PCIe x16.

À la fin de cette section, vous parviendrez à :

  • installer la carte réseau ;
  • installer la carte réseau sans fil ;
  • installer l’adaptateur vidéo.
