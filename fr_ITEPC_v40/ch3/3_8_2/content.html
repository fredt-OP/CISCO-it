3.8 Remontage des panneaux latéraux et connexion des câbles externes à l’ordinateur  
  3.8.2 Connexion des câbles externes à l’ordinateur  

Une fois les panneaux du boîtier remis en place, branchez les câbles à l’arrière de l’ordinateur. Voici quelques connexions de câbles externes courantes :

  • moniteur ;
  • clavier ;
  • souris ;
  • USB ;
  • ethernet ;
  • alimentation.

Lorsque vos connectez les câbles, assurez-vous qu’ils sont branchés aux bons emplacements sur l’ordinateur. Par exemple, certains câbles de souris et de clavier utilisent le même type de connecteur PS/2.

ATTENTION : lorsque vous branchez les câbles, ne forcez jamais une connexion.

REMARQUE : branchez le câble d’alimentation après avoir connecté tous les autres câbles.

Pour installer les câbles externes, procédez comme suit :

  1. Branchez le câble du moniteur sur le port vidéo. [Figure 1]
  2. Fixez le câble en serrant les vis sur le connecteur.
  3. Branchez le câble du clavier sur le port clavier PS/2. [Figure 2]
  4. Branchez le câble de la souris sur le port souris PS/2. [Figure 3]
  5. Branchez le câble USB sur le port USB. [Figure 4]
  6. Branchez le câble réseau sur le port réseau. [Figure 5]
  7. Branchez l’antenne sans fil sur le connecteur d’antenne. [Figure 6]
  8. Branchez le câble d’alimentation sur l’alimentation. [Figure 7]

La Figure 8 montre tous les câbles externes branchés à l’arrière de l’ordinateur.
