3.5 Installation des lecteurs dans les compartiments externes  

Les lecteurs, tels que les lecteurs optiques ou de disquette, sont installés dans des compartiments de lecteurs auxquels on accède par l’avant du boîtier. Les lecteurs optiques et les lecteurs de disquette stockent des données sur des supports amovibles. Les lecteurs situés dans les compartiments externes permettent d’accéder aux supports sans ouvrir le boîtier.

À la fin de cette section, vous parviendrez à :

  • installer le lecteur optique ;
  • installer le lecteur de disquette.
