12.2 Installation, configuration et optimisation d’un système d’exploitation  
  12.2.6 Description de l’installation, de l’utilisation et de la configuration d’un logiciel de messagerie  

Un logiciel de messagerie peut être installé comme un élément de navigateur Web ou sous forme d’application autonome. L’outil de messagerie Outlook Express est un composant du système d’exploitation Microsoft Windows. Pour configurer Outlook Express, vous devez fournir des informations sur votre compte de messagerie, comme illustré à la Figure 1.

Vous devez disposer des informations suivantes lorsque vous installez des comptes dans le logiciel client de messagerie :

  • nom complet ;
  • adresse de messagerie ;
  • type du serveur de messagerie pour courrier entrant, comme POP3 ou IMAP ;
  • nom du serveur de messagerie pour courrier entrant ;
  • nom du serveur de messagerie pour courrier sortant ;
  • nom du compte ;
  • mot de passe.

Les protocoles de messagerie utilisés comprennent :

  • Post Office Protocol version 3 (POP3) : récupère les courriels auprès d’un serveur distant, via TCP/IP. L’exemplaire du courriel enregistré sur le serveur est supprimé ; toutefois, certaines implémentations permettent aux utilisateurs de spécifier que ce courriel doit rester enregistré pour une durée déterminée.
  • Internet Message Access Protocol (IMAP) : permet aux clients de messagerie locaux de récupérer les courriels auprès d’un serveur. En général, il garde une copie du courriel sur le serveur, jusqu’à ce que vous placiez le courriel dans un dossier personnel de votre application de messagerie. Le protocole IMAP synchronise les dossiers de courriels entre le serveur et le client.
  • Simple Mail Transfer Protocol (SMTP) : transmet les courriels sur un réseau TCP/IP. Il s’agit du format de messagerie pour du texte utilisant uniquement un codage ASCII.
  • Multipurpose Internet Mail Extensions (MIME) : étend le format des courriels afin d’inclure du texte ASCII, ainsi que d’autres formats tels que des images et des documents de traitement de texte. Généralement utilisé conjointement avec le protocole SMTP.

Fonctions supplémentaires disponibles dans les logiciels de messagerie :

  • règles de traitement automatique des courriels ;
  • différents codages de courriels, tels que HTML, texte brut ou texte enrichi ;
  • forums de discussion.
