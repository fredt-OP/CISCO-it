12.3 Description de la mise à niveau des systèmes d’exploitation  

Un système d’exploitation doit être mis à niveau périodiquement pour rester compatible avec les dernières versions de matériel et de logiciels. À mesure que de nouvelles versions d’un système d’exploitation sont publiées, les systèmes plus anciens finissent par ne plus être pris en charge.

De nouveaux produits matériels arrivent constamment sur le marché. Ils exigent souvent l’installation du système d’exploitation le plus récent pour fonctionner correctement. Cela peut revenir cher, mais vous bénéficiez de fonctionnalités avancées grâce aux nouvelles fonctions et à la prise en charge de matériel plus récent.

Windows XP peut être mis à niveau à partir d’un CD ou d’un réseau. Vous devez vérifier que le nouveau système d’exploitation est compatible avec l’ordinateur. Microsoft fournit un utilitaire appelé Conseiller de mise à niveau, qui recherche des problèmes d’incompatibilité dans le système avant de mettre celui-ci à niveau vers un système d’exploitation Windows plus récent. Vous pouvez télécharger gratuitement le Conseiller de mise à niveau depuis le site Web de Microsoft Windows. Lorsque le Conseiller de mise à niveau a terminé, un rapport est généré, pour vous informer des problèmes éventuels. Une incompatibilité matérielle est la raison la plus fréquente de l’échec d’un processus de mise à niveau.

Tous les anciens systèmes d’exploitation Windows ne peuvent pas être mis à niveau vers des versions plus récentes, comme l’indique la liste suivante :

  • Windows 98, Windows 98 SE et Windows Me peuvent être mis à niveau vers Windows XP Édition Familiale ou Windows XP Professionnel.
  • Windows NT Workstation 4.0 avec Service Pack 6 et Windows 2000 Professionnel peuvent être mis à niveau uniquement vers Windows XP Professionnel.
  • Windows 3.1 et Windows 95 ne peuvent pas être mis à niveau vers Windows XP.

REMARQUE : n’oubliez pas de sauvegarder toutes les données avant de commencer une mise à niveau.
