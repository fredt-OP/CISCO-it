12.2 Installation, configuration et optimisation d’un système d’exploitation  

La plupart des systèmes d’exploitation sont faciles à installer. Une fois l’ordinateur démarré, le CD d’installation de Windows XP Professionnel affiche un assistant pour vous guider au cours de la procédure d’installation à l’aide d’une série de questions. Lorsque vous avez répondu aux questions, l’assistant d’installation termine l’installation automatiquement. Dans cette section, vous effectuerez une installation personnalisée de Windows XP Professionnel.

À la fin de cette section, vous parviendrez à :

  • comparer une installation par défaut et une installation personnalisée ;
  • installer Windows XP Professionnel en utilisant une installation personnalisée ;
  • créer, visualiser et gérer les disques, les répertoires et les fichiers ;
  • identifier les procédures et les utilitaires utilisés pour optimiser les performances des systèmes d’exploitation ;
  • identifier les procédures et les utilitaires utilisés pour optimiser les performances des navigateurs ;
  • décrire l’installation, l’utilisation et la configuration d’un logiciel de messagerie ;
  • définir la résolution d’un écran et mettre à jour le pilote vidéo ;
  • décrire l’installation d’un deuxième système d’exploitation.
