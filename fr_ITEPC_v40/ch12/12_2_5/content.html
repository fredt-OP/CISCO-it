12.2 Installation, configuration et optimisation d’un système d’exploitation  
  12.2.5 Identification des procédures et des utilitaires d’optimisation des performances des navigateurs  

Les navigateurs Web et la messagerie électronique sont généralement les applications les plus utilisées sur un ordinateur. L’optimisation des navigateurs Web et de la messagerie devrait donc améliorer les performances de l’ordinateur.

Le navigateur Microsoft, Internet Explorer (IE), dispose de paramètres généraux pour modifier sa page d’accueil et son apparence. Des paramètres supplémentaires vous permettent de visualiser ou de supprimer les informations enregistrées par le navigateur :

  • historique ;
  • fichiers temporaires ;
  • cookies ;
  • mots de passe ;
  • informations de formulaire Web.

REMARQUE : les cookies sont des informations transmises entre un navigateur Web et un serveur Web, dans l’objectif de conserver une trace des informations sur l’utilisateur, afin de personnaliser la page fournie à ce dernier.

Pour accéder aux paramètres d’IE, ouvrez une fenêtre de navigateur IE, puis choisissez Outils > Options Internet.

La mise en cache, ou stockage, des fichiers Internet est une fonction du navigateur Web utilisée pour accélérer l’accès à des sites Web précédemment visités. L’outil de stockage des fichiers dans IE télécharge sur le disque dur des copies des images ou des fichiers HTML des sites visités. Lorsque vous visitez à nouveau l’un de ces sites, celui-ci s’ouvre plus rapidement parce que les fichiers sont dans le cache sur le disque local et il n’est pas nécessaire de les télécharger à nouveau.

Les fichiers présents dans le cache du navigateur Web peuvent se périmer ou être très lourds. Ces paramètres IE vous permettent de contrôler la taille du cache et les circonstances dans lesquelles il doit être actualisé :

  • à chaque visite de la page ;
  • à chaque démarrage d’IE ;
  • automatiquement ;
  • jamais.

Pour accéder aux paramètres du cache, ouvrez une fenêtre de navigateur IE, puis choisissez Outils > Options Internet. Dans la zone Fichiers Internet temporaires, cliquez sur Paramètres. Cliquez sur les onglets dans la Figure 1 pour explorer d’autres options de configuration d’IE.
