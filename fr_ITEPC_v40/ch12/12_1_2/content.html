  12.1.2 Description des systèmes d’exploitation de réseau  

Un système d’exploitation de réseau est un système d’exploitation contenant des fonctions supplémentaires pour améliorer la fonctionnalité et la gestion d’un environnement de réseau. Voici quelques exemples de systèmes d’exploitation de réseau :

  • Windows 2000 Server
  • Windows 2003 Server
  • UNIX
  • Linux
  • Novell NetWare
  • Mac OS X

Un système d’exploitation de réseau est conçu pour fournir des ressources réseau aux clients :

  • applications serveur, par exemple des bases de données partagées ;
  • stockage de données centralisé ;
  • services d’annuaire fournissant un référentiel centralisé de comptes d’utilisateur et de ressources sur le réseau, comme LDAP ou Active Directory ;
  • file d’attente d’impression réseau ;
  • accès au réseau et sécurité du réseau ;
  • systèmes de stockage redondants, comme la technologie RAID et les sauvegardes.

Les systèmes d’exploitation de réseau fournissent plusieurs protocoles conçus pour réaliser des fonctions réseau. Ces protocoles sont contrôlés par du code sur les serveurs réseau. Les protocoles utilisés par les systèmes d’exploitation réseau fournissent des services tels que la navigation sur le Web, le transfert de fichier, les courriels, la résolution de nom et l’adressage IP automatique.
