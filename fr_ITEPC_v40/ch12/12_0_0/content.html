12.0 Introduction  

L’installation, la configuration et l’optimisation des systèmes d’exploitation sont examinés plus en détail dans ce chapitre.

Diverses marques de systèmes d’exploitation sont disponibles aujourd’hui sur le marché, notamment Microsoft Windows, Apple Mac OS, UNIX et Linux. Un technicien doit prendre en considération le système actuel de l’ordinateur lorsqu’il sélectionne un système d’exploitation. En outre, un système d’exploitation comporte plusieurs versions ou distributions. Voici quelques versions de Microsoft Windows : Windows 2000 Professionnel, Windows XP Édition Familiale, Windows XP Professionnel, Windows Media Center, Windows Vista Édition Familiale Basique, Windows Vista Entreprise et Windows Vista Premium.

Chacun de ces systèmes d’exploitation propose de nombreuses fonctionnalités identiques, avec une interface similaire. Toutefois, certaines fonctions nécessaires à un client donné peuvent ne pas être disponibles sur tous ces systèmes. Vous devez donc être capable de comparer les systèmes d’exploitation, afin de trouver celui qui répond le mieux aux besoins de votre client.

À la fin de ce chapitre, vous parviendrez à :

  • sélectionner le système d’exploitation en fonction des besoins d’un client ;
  • installer, configurer et optimiser un système d’exploitation ;
  • décrire comment mettre à niveau des systèmes d’exploitation ;
  • décrire des procédures de maintenance préventive pour les systèmes d’exploitation ;
  • dépanner des systèmes d’exploitation.
