12.2 Installation, configuration et optimisation d’un système d’exploitation  
  12.2.1 Comparaison d’une installation par défaut et d’une installation personnalisée  

L’installation par défaut de Windows XP Professionnel est suffisante pour la plupart des ordinateurs utilisés dans un réseau domestique ou celui d’une petite entreprise. En général, l’installation personnalisée de Windows XP Professionnel est utilisée dans un réseau de plus grandes dimensions.

Installation par défaut
L’installation par défaut nécessite une intervention minimale de l’utilisateur. Vous êtes invité à fournir des informations sur l’ordinateur concerné et sur le propriétaire/l’utilisateur.

Installation personnalisée
Sous Windows XP, l’installation personnalisée est très similaire à l’installation par défaut. Deux écrans seulement proposent une sélection personnalisée durant l’installation. Le premier écran permet de personnaliser les paramètres régionaux et le second les paramètres réseau, comme illustré à la Figure 1. Généralement, un technicien ou un utilisateur techniquement expérimenté effectue l’installation personnalisée. Dans une installation personnalisée, l’assistant demande à l’utilisateur des informations de performances détaillées, afin de garantir que le système d’exploitation est personnalisé pour répondre aux préférences ou exigences de l’utilisateur individuel ou de l’administrateur réseau d’une entreprise. Vous pouvez installer Windows XP Professionnel de façon personnalisée sur plusieurs ordinateurs sur un réseau, en utilisant un fichier de réponses qui contient des réponses et des paramètres prédéfinis aux questions posées par l’assistant au cours de l’installation.

Le technicien peut automatiser et personnaliser une installation de Windows XP Professionnel pour qu’elle comprenne les fonctions suivantes :

  • applications de productivité, comme Microsoft Office ;
  • applications personnalisées ;
  • prise en charge de plusieurs langues ;
  • OS Deployment Feature Pack utilisant Microsoft Systems Management Server (SMS) ;
  • pilotes de périphériques matériel.
