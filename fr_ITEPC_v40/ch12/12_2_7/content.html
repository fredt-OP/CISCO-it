12.2 Installation, configuration et optimisation d’un système d’exploitation  
  12.2.7 Définition de la résolution d’un écran et mise à jour d’un pilote vidéo  

Une fois le système d’exploitation installé, vous pouvez définir la résolution de l’écran afin de répondre aux besoins de votre client. Si cette résolution n’est pas correctement définie, vous pouvez obtenir des différentes cartes et moniteurs vidéo, des résultats surprenants. Par exemple, le bureau Windows peut ne pas occuper toute la surface de l’écran ou l’écran peut être vide si la résolution est trop élevée.

Lorsque vous utilisez un écran à cristaux liquides, la résolution doit être définie sur mode natif ou résolution native. Le mode natif correspond à une résolution d’écran égale au nombre de pixels du moniteur. Si vous vous éloignez du mode natif, le moniteur ne produit plus la meilleure image. Reportez-vous à la Figure 1 pour les paramètres de résolution d’écran d’un ordinateur sous Windows XP Professionnel.

Vous pouvez modifier les paramètres de l’écran dans l’onglet Paramètres accessible via l’applet Affichage du panneau de configuration :

  • Résolution de l’écran : détermine le nombre de pixels. Plus le nombre de pixels est élevé, plus la résolution et l’image sont bonnes.
  • Fréquence d’actualisation : détermine la fréquence à laquelle l’image est réaffichée sur l’écran. La fréquence d’actualisation est exprimée en Hertz (Hz). Plus elle est élevée, plus l’image est stable.
  • Qualité couleur : détermine le nombre de couleurs simultanément visibles à l’écran. Les couleurs sont créées en variant l’intensité de trois couleurs de base (rouge, vert et bleu). Plus le nombre de bits est élevé, plus le nombre de couleurs est élevé. Voici une liste de profondeurs d’échantillonnage :
    • 256 couleurs : couleur 8 bits ;
    • 65 536 couleurs : couleur 16 bits  (milliers de couleurs) ;
    • 16 millions de couleurs : couleur 24 bits  (couleurs réelles) ;
    • 16 millions de couleurs : 24 bits (couleurs réelles avec remplissage à 8 bits pour permettre un traitement 32 bits).

En cas de dépannage d’un problème d’affichage, vérifiez que le pilote est totalement compatible avec la carte graphique. Windows peut installer un pilote par défaut qui fonctionne, mais qui ne fournit pas toutes les options disponibles pour le meilleur affichage et les meilleures performances. La Figure 2 illustre l’utilitaire de mise à jour du pilote vidéo sous Windows XP Professionnel. Pour obtenir les meilleures performances graphiques, procédez comme suit :

  1. Téléchargez le pilote le plus récent depuis le site Web du fabricant.
  2. Supprimez le pilote actuel.
  3. Désactivez le logiciel antivirus.
  4. Installez le nouveau pilote.
  5. Redémarrez l’ordinateur.

REMARQUE : lorsque vous désactivez le logiciel antivirus, votre ordinateur devient vulnérable ; vous ne devez donc pas le faire si vous être connecté à Internet.

Des problèmes peuvent survenir lorsque vous installez ou réinstallez un pilote vidéo. Par exemple, une fois les étapes de performances graphiques effectuées, l’écran ne s’affiche pas lorsque vous redémarrez l’ordinateur. Pour résoudre ce problème et rétablir les paramètres, redémarrez l’ordinateur. Pendant la phase d’amorçage, appuyez sur la touche F8. Lorsque vous y êtes invité, entrez les options de démarrage, puis sélectionnez Activer le mode VGA pour utiliser une résolution 640 x 480. Une fois le système d’exploitation chargé, vous pouvez sélectionner Revenir à la version précédente dans les Propriétés de la carte graphique. Vous pouvez ensuite rechercher les problèmes possibles du pilote que vous avez tenté d’installer.
