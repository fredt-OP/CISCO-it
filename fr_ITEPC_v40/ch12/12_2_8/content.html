12.2 Installation, configuration et optimisation d’un système d’exploitation  
  12.2.8 Description de l’installation d’un deuxième système d’exploitation  

Vous pouvez disposer de plusieurs systèmes d’exploitation sur un seul ordinateur. Certaines applications exigent la version la plus récente d’un système d’exploitation, alors que d’autres ont besoin d’une version antérieure. Un processus de double amorçage existe pour gérer plusieurs systèmes d’exploitation sur un ordinateur. Lorsque le fichier boot.ini détermine, au cours du processus d’amorçage, que plusieurs systèmes d’exploitation sont présents, vous êtes invité à choisir le système d’exploitation que vous souhaitez charger. La Figure 1 présente un exemple de fichier boot.ini.

Configuration d’un double amorçage
Pour créer un système à double amorçage sous Microsoft Windows, vous devez généralement disposer de plusieurs disques durs ou le disque dur doit comporter plus d’une partition.

Vous devez installer le système d’exploitation le plus ancien sur la partition primaire ou sur le disque dur marqué comme première partition active. Vous devez ensuite installer le deuxième système d’exploitation sur la deuxième partition ou le deuxième disque dur. Les fichiers d’amorce sont installés automatiquement dans la partition active.

Pendant l’installation, le fichier boot.ini est créé sur la partition active, afin de permettre la sélection du système d’exploitation à amorcer au démarrage. Le fichier boot.ini peut être modifié pour inverser l’ordre des systèmes d’exploitation. Dans ce fichier, vous pouvez également modifier la durée, au cours de la phase d’amorçage, pendant laquelle la sélection du système d’exploitation est possible. Généralement, la durée par défaut de sélection d’un système d’exploitation est de 30 secondes. Cela retarde toujours l’amorce de l’ordinateur de 30 secondes, sauf si l’utilisateur intervient pour sélectionner un système d’exploitation particulier. Dans le fichier boot.ini, vous pouvez ramener cette durée à 5 ou 10 secondes pour que l’ordinateur démarre plus rapidement.

Pour modifier le fichier boot.ini, cliquez avec le bouton droit de la souris sur Poste de travail > Propriétés > onglet Avancé. Dans la zone Démarrage et récupération, sélectionnez Paramètres. Cliquez sur Modifier.
